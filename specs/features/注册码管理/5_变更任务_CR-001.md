## 3. 变更详情

### 修改的验收标准状态

- **AC-001 ~ AC-011, AC-016, AC-017, AC-020**: 前端注册码管理入口暂不开放
  - **Given** 用户进入平台管理 UI
  - **When** 查看侧边栏导航或尝试访问注册码管理入口
  - **Then** 不再看到「注册码管理」入口，且前端不注册 `/registration-codes` 页面路由

### 技术变更

- 移除 `frontend/src/App.tsx` 中 `RegistrationCodeList` 的 import 和 route 注册。
- 移除 `frontend/src/components/Sidebar.tsx` 中 `/registration-codes` 菜单项及专属 icon 渲染分支。
- 移除 `frontend/src/layouts/MainLayout.tsx` 中注册码导航配置。
- 不修改后端 registration 模块、Prisma schema 和 nodered-plugin 注册流程。

### 兼容性说明

本 CR 仅下线前端入口，不删除后端 API、数据库模型或插件注册字段，因此不会破坏已有网关注册链路。未来需要恢复时，可重新挂载前端路由和菜单入口。
