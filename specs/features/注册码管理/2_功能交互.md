# 注册码管理 - 页面功能交互说明

## 1. 页面综述

- **功能描述**：管理网关接入平台的注册码，支持批量生成、状态追踪、作废和删除操作。
- **页面类型**：后台管理类（Admin / Table-First）
- **目标用户**：IoT 系统运维人员
- **核心目标**：安全可控地管理网关接入凭证的生命周期
- **本阶段范围**：注册码列表页、批量生成弹窗、详情查看弹窗、作废/删除操作

## 2. 页面结构

- **页面标题**：注册码管理
- **主体布局**：左侧筛选区 + 右侧列表展示区
- **主要页面/区域**：
  - 注册码列表页（主页面）
  - `批量生成` 弹窗
  - `注册码详情` 弹窗（已使用注册码查看）
  - `作废确认` 弹窗（已使用时）
  - `删除确认` 弹窗

## 3. 注册码列表页

### 3.1 筛选区

- **搜索字段**：
  - `注册码`：输入框，模糊搜索
  - `网关名称`：输入框，模糊搜索
  - `状态`：下拉选，默认「全部」
    - 选项：全部、未使用、已使用、已过期、已作废
- **交互**：
  - 点击 `查询`：更新列表数据，重置页码为 1
  - 点击 `重置`：清空所有筛选条件

### 3.2 操作按钮区

- **`批量生成`**：
  - **交互**：打开 `批量生成注册码（弹窗）`

### 3.3 列表展示区

- **排序规则**：默认按 `生成时间` 倒序排列
- **分页规则**：默认 20 条/页，支持切换 10/20/50 条/页
- **加载状态**：接口请求时，列表区域显示 Loading 骨架屏
- **空态**：无数据时显示「暂无注册码，请点击上方按钮生成」
- **列表字段**：
  - `注册码`：格式为 `A1B2C3D4-********`（后 8 位隐藏），已使用的注册码整行可点击
  - `状态`：Tag 标签展示
    - 未使用：蓝色
    - 已使用：绿色
    - 已过期：灰色
    - 已作废：红色
  - `生成时间`：格式 `YYYY-MM-DD HH:mm`，精确到分钟
  - `过期时间`：格式 `YYYY-MM-DD HH:mm`，精确到分钟；永不过期显示 `-`
  - `使用时间`：格式 `YYYY-MM-DD HH:mm`，仅已使用时显示，未使用显示 `-`
  - `绑定网关`：已使用时显示网关名称，未使用显示 `-`
- **行内操作**：
  - 未使用：`作废`
  - 已使用：`作废`（需确认）
  - 已过期：无操作按钮
  - 已作废：`删除`

### 3.4 批量生成注册码（弹窗）

- **打开方式**：点击列表页 `批量生成` 按钮
- **关闭方式**：点击 `×` 或点击遮罩层
- **表单字段**：
  - `生成数量`：数字输入框，必填
    - 范围：1-50
    - 默认值：1
    - 校验：超出范围时提示「单次最多生成 50 个注册码」
  - `有效期`：下拉选，必填
    - 选项：7 天、30 天、90 天、180 天、永不过期、自定义
    - 默认值：30 天
    - 当选择「自定义」时：显示 `天数` 输入框，范围 1-3650，提示「有效期需在 1-3650 天之间」
- **操作按钮**：
  - `取消`：关闭弹窗
  - `生成`：提交表单，生成注册码
- **交互规则**：
  - 点击 `生成` 后，按钮显示 Loading 状态并置灰
  - 生成成功后，弹窗关闭，列表刷新显示新生成的注册码，Toast 提示「成功生成 N 个注册码」
  - 生成失败时，按钮恢复，Toast 提示错误原因

### 3.5 注册码详情（弹窗）

- **打开方式**：点击已使用的注册码行
- **显示内容**：
  - `注册码`：完整显示（如 `A1B2C3D4-E5F6G7H8`）
  - `状态`：Tag 标签
  - `绑定网关`：网关名称
  - `注册时间`：精确到分钟
  - `注册IP`：网关注册时的 IP 地址
- **操作按钮**：
  - `关闭`：关闭弹窗

### 3.6 作废确认（弹窗）

- **打开方式**：点击已使用注册码的 `作废` 按钮
- **弹窗类型**：Confirm 对话框
- **显示内容**：
  - 标题：作废确认
  - 提示文案：「作废后，该网关将无法继续上报数据。确定要作废此注册码吗？」
  - `注册码`：`A1B2C3D4-********`
  - `绑定网关`：[网关名称]
- **操作按钮**：
  - `取消`：关闭弹窗，保持注册码状态不变
  - `确认作废`：执行作废操作

### 3.7 删除确认（弹窗）

- **打开方式**：点击已作废注册码的 `删除` 按钮
- **弹窗类型**：Confirm 对话框
- **显示内容**：
  - 标题：删除确认
  - 提示文案：「确定要删除此注册码吗？删除后无法恢复。」
  - `注册码`：`A1B2C3D4-********`
- **操作按钮**：
  - `取消`：关闭弹窗，保持注册码记录
  - `确认删除`：执行删除操作

## 4. 状态流转与操作映射

| 当前状态 | 可执行操作 | 操作后状态 |
|---------|-----------|-----------|
| 未使用 | `作废` | 已作废 |
| 已使用 | `作废`（二次确认） | 已作废 |
| 已过期 | — | — |
| 已作废 | `删除` | 移除列表 |

## 5. 异常与边界处理

| 场景 | 处理方式 |
|------|----------|
| 注册码已被使用 | 其他网关尝试使用同一注册码时，返回「注册码已被使用」 |
| 注册码已过期 | nodered-plugin 验证时返回「注册码已过期」 |
| 注册码已作废 | nodered-plugin 验证时返回「注册码已作废」 |
| 注册码不存在 | nodered-plugin 验证时返回「注册码不存在」 |
| 超过有效期范围 | 表单校验提示「有效期需在 1-3650 天之间」 |
| 超过数量上限 | 表单校验提示「单次最多生成 50 个注册码」 |
| 删除失败 | Toast 提示错误原因 |
