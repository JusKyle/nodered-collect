## 2. 数据模型设计

### 2.1 Prisma Schema

```prisma
model DeviceModel {
  id          String        @id @default(cuid())
  name        String
  vendor      String
  model       String
  protocol    String
  description String?
  status      ModelStatus   @default(ENABLED)
  points      Json          @default("[]")
  version     String        @default("v1.0")
  createdAt   DateTime      @default(now())
  updatedAt   DateTime      @updatedAt

  deviceInstances DeviceInstance[]
  modelVersions   ModelVersion[]

  @@unique([name, vendor, model])
  @@index([status])
  @@index([protocol])
}

model ModelVersion {
  id          String      @id @default(cuid())
  modelId     String
  version     String
  points      Json
  changelog   String?
  createdAt   DateTime    @default(now())

  model       DeviceModel @relation(fields: [modelId], references: [id])

  @@unique([modelId, version])
}

enum ModelStatus {
  ENABLED
  DISABLED
}
```

### 2.2 Point 数据结构

```typescript
interface Point {
  id: string
  name: string
  code: string
  dataType: 'int' | 'float' | 'string' | 'bool' | 'date' | 'datetime' | 'json'
  access: 'read' | 'write' | 'readWrite'
  address: string
  offset?: number
  length?: number
  scanInterval: number
  timeout?: number
  unit?: string
  transform?: string
  alarmMin?: number
  alarmMax?: number
}
```

## 3. API 设计

### 3.1 设备模型 API

#### GET /api/device-models
获取模型列表。

**查询参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 否 | 模型名称模糊匹配 |
| protocol | string | 否 | 协议类型筛选 |
| status | string | 否 | ENABLED/DISABLED |
| page | number | 否 | 页码，默认 1 |
| pageSize | number | 否 | 每页条数，默认 20 |

**响应**
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "id": "clx123",
        "name": "西门子 S7-1200",
        "vendor": "西门子",
        "model": "S7-1200",
        "protocol": "S7",
        "description": "标准 PLC",
        "status": "ENABLED",
        "version": "v1.1",
        "pointCount": 15,
        "usageCount": 3,
        "createdAt": "2026-06-17T08:00:00Z"
      }
    ],
    "pagination": { "total": 20, "page": 1, "pageSize": 20 }
  }
}
```

→ AC-010, AC-011

#### POST /api/device-models
创建设备模型。

**请求**
```json
{
  "name": "西门子 S7-1200",
  "vendor": "西门子",
  "model": "S7-1200",
  "protocol": "S7",
  "description": "标准 PLC",
  "points": [
    { "name": "温度", "code": "temp_001", "dataType": "float", "access": "read", "address": "40001", "scanInterval": 1000 }
  ]
}
```

**响应**
```json
{ "success": true, "data": { "id": "clx123", "version": "v1.0", "pointCount": 1 } }
```

→ AC-001, AC-002, AC-003, AC-014, AC-015, AC-016

#### PUT /api/device-models/:id
更新设备模型（创建新版本）。

**行为**
1. 获取当前版本号
2. 将当前点位存入 ModelVersion
3. 递增版本号（v1.0 → v1.1）
4. 更新 DeviceModel 的 points 和 version

**响应**
```json
{ "success": true, "data": { "version": "v1.1" } }
```

→ AC-012, AC-023

#### POST /api/device-models/:id/duplicate
复制设备模型。

**响应**
```json
{
  "success": true,
  "data": { "id": "clx456", "name": "西门子 S7-1200_副本", "version": "v1.0", "pointCount": 15 }
}
```

→ AC-006, AC-021

#### PUT /api/device-models/:id/status
切换模型状态。

**请求**
```json
{ "status": "DISABLED" }
```

→ AC-008, AC-009, AC-020

**行为**
1. 检查 usageCount > 0 → 返回错误

## 4. 核心逻辑设计

### 4.1 版本递增逻辑

```typescript
// device-model.service.ts
async updateModel(id: string, dto: UpdateDeviceModelDto) {
  const model = await prisma.deviceModel.findUnique({ where: { id } });

  await prisma.modelVersion.create({
    data: {
      modelId: id,
      version: model.version,
      points: model.points,
    }
  });

  const versionParts = model.version.replace('v', '').split('.');
  const newVersion = `v${versionParts[0]}.${parseInt(versionParts[1]) + 1}`;

  return prisma.deviceModel.update({
    where: { id },
    data: { ...dto, version: newVersion }
  });
}
```

→ AC-012, AC-023

### 4.2 点位编码唯一性校验

```typescript
// device-model.dto.ts
export const createDeviceModelDto = z.object({
  points: z.array(z.object({
    name: z.string(),
    code: z.string(),
  }))
}).refine(
  (data) => {
    const codes = data.points?.map(p => p.code) || [];
    return new Set(codes).size === codes.length;
  },
  { message: "点位编码不能重复" }
);
```
