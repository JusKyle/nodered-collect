# 设备实例管理 - 页面功能交互说明

## 1. 页面综述

- **功能描述**：管理基于设备模板创建的设备实例，支持实例创建、通信参数配置、网关分配、点位管理、配置下发和删除，提供实时数据展示和历史数据查询。
- **页面类型**：后台管理类 (Admin / Table-First) + 详情页
- **目标用户**：工业自动化工程师、IoT 系统运维人员、数据采集系统管理员
- **核心目标**：用户可以在平台上完成设备实例的全生命周期管理，实时查看采集数据和下发状态，无需登录网关操作。
- **本阶段范围**：实例创建（单个+批量导入）、通信参数配置、网关分配/变更、点位管理、同步模板版本、配置下发、删除实例、实例列表、列表筛选、实例详情页、实时值展示、历史数据查询

---

## 2. 页面结构

- **页面标题**：设备实例
- **页面副标题**：管理所有设备实例的创建、配置和数据采集
- **主体布局**：导航栏 + 顶部操作区 + 数据表格区
- **主要页面/区域**：
  - 导航栏：采集网关，注册码，设备模型，设备实例，配置记录，系统配置
  - 实例列表页（主页面）
  - 新建设备实例（弹窗）
  - 批量导入（弹窗）
  - 配置通信参数（抽屉）
  - 分配网关（弹窗）
  - 批量分配网关（弹窗）
  - 变更网关（弹窗）
  - 批量变更网关（弹窗）
  - 点位管理（抽屉）
  - 同步模板版本确认（弹窗）

## 3. 实例列表页

### 3.1 搜索/筛选区

- **筛选字段**：
  - `分组`：下拉选，默认`全部`，选项为已有的设备分组列表
  - `模板名称`：下拉选，默认`全部`，选项为已有的设备模板列表
  - `网关名称`：下拉选，默认`全部`，选项为已有的网关列表
  - `状态`：下拉选，默认`全部`，选项：`全部`、`在线`、`离线`、`采集中`
- **交互**：
  - 选择筛选条件后需点击`查询`按钮
  - 多个筛选条件叠加生效
  - 点击`重置`按钮清空所有筛选条件，恢复默认状态

### 3.2 操作按钮区

- **`新增实例`**：主按钮（primary），点击打开`新建设备实例（弹窗）`
- **`批量导入`**：次按钮（default），点击打开`批量导入（弹窗）`
- **`批量分配网关`**：次按钮（default），当选中列表中至少一条记录时启用，点击打开`批量分配网关（弹窗）`
- **`批量变更网关`**：次按钮（default），当选中列表中至少一条记录时启用，点击打开`批量变更网关（弹窗）`
- **`批量下发`**：次按钮（default），当选中列表中至少一条记录时启用，点击触发批量下发

### 3.3 列表展示区

- **多选框**：支持多选，表头全选/反选
- **排序规则**：默认按 `创建时间` 倒序排列。
- **列表字段**：
  - `实例名称`：设备实例名称，加粗显示，可点击跳转到实例详情页
  - `模板名称`：所关联的设备模板名称，无数据显示 `-`
  - `分组`：所属分组名称，无数据显示 `-`
  - `网关名称`：所分配的网关名称，未分配显示 `未分配`
  - `状态`：使用 StatusBadge 展示，映射关系：
    - `ONLINE` → 在线（绿）
    - `OFFLINE` → 离线（灰）
    - `COLLECTING` → 采集中（蓝）
- **行内操作**：
  - `配置通信参数`：点击打开`配置通信参数（抽屉）`
  - `分配网关`：未分配网关时显示，点击打开`分配网关（弹窗）`
  - `变更网关`：已分配网关时显示，点击打开`变更网关（弹窗）`
  - `点位管理`：点击打开`点位管理（抽屉）`
  - `同步模板版本`：当模板有新版本时显示，点击打开`同步模板版本确认（弹窗）`
  - `下发配置`：点击触发单实例下发配置
  - `删除`：点击打开`删除确认（弹窗）`

## 4. 新建设备实例（弹窗）

- **打开方式**：点击列表页 `新增实例` 按钮
- **弹窗宽度**：中等（约 520px）
- **表单字段**：
  - `设备模板`：必填，下拉选，选项为已有的设备模板列表
  - `设备ID`：必填，输入框，最多 100 字符，全局唯一
  - `实例名称`：必填，输入框，最多 100 字符
  - `分组`：选填，下拉选，选项为已有的设备分组列表
- **底部操作**：
  - `取消`：关闭弹窗，重置表单
  - `确认创建`：
    - 按钮变为「创建中...」并禁用
    - 调用创建实例 API
    - 创建成功后刷新列表，关闭弹窗，Toast 提示「创建成功」
- **校验规则**：
  - 实时校验（输入时）+ 提交时全量校验
  - `设备ID` 唯一性校验
  - 错误提示：输入框红框 + 下方红色错误文字

---

## 5. 批量导入（弹窗）

- **打开方式**：点击列表页 `批量导入` 按钮
- **弹窗宽度**：中等（约 520px）
- **导入说明**：
  - 支持 Excel 格式（.xlsx, .xls）
  - 文件包含字段：设备ID、实例名称、分组、模板ID、网关ID、通信参数（IP、端口、从站地址等）
  - 提供`下载模板`链接，下载标准 Excel 模板
- **上传区域**：
  - 拖拽或点击上传文件
  - 支持格式：.xlsx, .xls
  - 文件大小限制：`待补充`
- **校验规则**：
  - 上传后系统自动校验
  - 重复设备ID：导入失败，提示重复的设备ID列表
  - 模板ID不存在：导入失败，提示不存在的模板ID
  - 网关ID不存在：导入失败，提示不存在的网关ID
- **导入结果**：
  - 成功：Toast 提示「成功导入 {N} 条」，刷新列表，关闭弹窗
  - 失败：弹窗内显示错误详情（具体错误项和原因），不关闭弹窗
- **底部操作**：
  - `取消`：关闭弹窗
  - `确认导入`：选择文件后启用，点击开始导入

---

## 6. 配置通信参数（抽屉）

- **打开方式**：点击列表行内 `配置通信参数` 按钮
- **抽屉方向**：右侧滑出
- **抽屉宽度**：宽（约 600px）
- **表单字段**：
  - 通信参数根据所选设备模板的协议类型动态显示
  - 通用字段：
    - `IP地址`：必填，输入框，合法 IP 或域名格式校验
    - `端口`：必填，数字输入框，范围 1-65535
  - 协议特有字段（根据模板协议类型动态显示）：
    - `从站地址`：Modbus 协议时显示，数字输入框，范围 1-255
    - `待补充`：其他协议参数
- **底部操作**：
  - `取消`：关闭抽屉，不保存
  - `保存`：
    - 按钮变为「保存中...」并禁用
    - 调用保存通信参数 API
    - 成功后 Toast 提示「保存成功」，关闭抽屉，刷新列表

## 9. 点位管理（抽屉）

- **打开方式**：点击列表行内 `点位管理` 按钮
- **抽屉方向**：右侧滑出
- **抽屉宽度**：宽（约 800px）
- **Tab 切换**：
  - `继承点位`：从模板继承的点位列表
  - `设备级点位`：在设备上新增的点位列表

### 9.1 继承点位 Tab

- **列表字段**：
  - `点位名称`：点位名称
  - `地址`：寄存器地址
  - `数据类型`：数据类型
  - `采集间隔`：采集间隔（秒）
  - `来源`：固定显示 `模板级`
- **行内操作**：
  - `编辑`：点击后行内变为可编辑状态，可修改点位属性（如采集间隔、数据类型等）
  - `保存`：编辑状态下显示，点击保存修改
  - `取消`：编辑状态下显示，点击取消编辑
- **说明**：修改继承点位属性后，该点位仍属于模板级；同步模板版本时，修改会被模板最新版本覆盖

### 9.2 设备级点位 Tab

- **顶部操作**：
  - `新增点位`：主按钮，点击新增一行空白点位表单
- **列表字段**：
  - `点位名称`：必填，输入框
  - `地址`：必填，输入框，寄存器地址
  - `数据类型`：必填，下拉选，选项根据协议类型确定
  - `采集间隔`：必填，数字输入框，单位：秒
  - `来源`：固定显示 `设备级`
- **行内操作**：
  - `删除`：点击删除该行点位
- **底部操作**：
  - `取消`：关闭抽屉，不保存
  - `保存`：
    - 按钮变为「保存中...」并禁用
    - 调用保存点位 API
    - 成功后 Toast 提示「保存成功」，关闭抽屉

## 11. 下发配置

### 11.1 单个下发

- **触发方式**：点击列表行内 `下发配置` 按钮
- **下发逻辑**：
  - 系统比对配置版本号和已下发版本号
  - 有变化则全量下发到网关 Node-RED
- **状态反馈**：
  - 下发中：按钮变为「下发中...」并禁用
  - 成功：Toast 提示「下发成功」，刷新列表
  - 失败：Toast 提示具体失败原因
  - 网关离线：Toast 提示「网关离线，下发失败」

### 11.2 批量下发

- **触发方式**：选中多条记录后点击顶部 `批量下发` 按钮
- **确认弹窗**：
  - 标题：「确认下发配置？」
  - 内容：已选择 {N} 个实例，确认下发配置到对应网关？
  - 操作：`取消`、`确认下发`
- **下发逻辑**：同单个下发，批量执行
- **结果反馈**：
  - 全部成功：Toast 提示「成功下发 {N} 个实例」
  - 部分失败：Toast 提示「成功 {N} 个，失败 {M} 个」，可查看详情
  - 全部失败：Toast 提示具体失败原因

- **打开方式**：点击列表行内 `删除` 按钮
- **弹窗宽度**：中等（约 448px）
- **弹窗内容**：
  - 标题：「确定删除实例「{实例名称}」吗？」
  - 辅助说明：删除实例将从网关移除配置并删除平台记录，此操作不可恢复
- **底部操作**：

  - `确认删除`：
    - 按钮变为「删除中...」并禁用

    - 网关在线：先从网关删除配置，再删除平台记录
    - 网关离线：提示「网关离线，无法删除配置」，实例记录仍保留
    - 成功后 Toast 提示「删除成功」，关闭弹窗，刷新列表

## 13. 实例详情页

- **入口**：点击列表页实例名称，或通过路由 `/instances/:id` 访问
- **返回**：左上角返回箭头，返回实例列表页

### 13.1 页头区

- **实例名称**：大标题
- **状态徽章**：在线/离线/采集中
- **关键信息**：模板名称、分组、网关名称
- **操作按钮**：`配置通信参数`、`分配网关`/`变更网关`、`点位管理`、`下发配置`、`删除`

### 13.2 通信参数卡片

- **布局**：两列键值对
- **字段**：
  - `IP地址`
  - `端口`
  - `从站地址`（Modbus 协议时显示）
  - `协议类型`
  - `其他参数`（根据协议类型动态显示）
- **操作**：右上角`编辑`按钮，点击打开`配置通信参数（抽屉）`

### 13.3 网关信息卡片

- **布局**：两列键值对
- **字段**：
  - `网关名称`：可点击跳转到网关详情页
  - `网关状态`：在线/离线
  - `NR版本`
  - `最后心跳`
- **操作**：
  - 未分配网关：显示`分配网关`按钮，点击打开`分配网关（弹窗）`
  - 已分配网关：显示`变更网关`按钮，点击打开`变更网关（弹窗）`

### 13.4 采集状态卡片

- **状态展示**：
  - 当前状态：在线/离线/采集中（大号 StatusBadge）
  - 最后数据时间：最近一次采集数据的时间
  - 点位总数：总点位数量
  - 采集正常点位数：正常采集的点位数量
- **说明**：
  - 在线：网关在线且能通信
  - 离线：网关离线或不能通信
  - 采集中：在线且能采集到点位数值

### 13.5 点位实时值卡片

> 对应需求：AC-015、AC-017、AC-019

- **列表展示**：所有点位的实时采集值
- **列表字段**：
  - `点位名称`
  - `当前值`：实时值，通过 SSE 推送更新
  - `单位`：点位单位，无数据显示 `-`
  - `更新时间`：最近一次更新时间，格式 `YYYY-MM-DD HH:mm:ss`
  - `来源`：模板级 / 设备级
- **实时更新机制**：
  - **SSE 订阅**：页面挂载时建立 SSE 连接，订阅该实例的点位实时值
  - **更新触发**：收到新的采集数据时，更新对应点位的当前值和更新时间
  - **断线重连**：SSE 连接断开时自动重试（指数退避）
- **历史数据入口**：行内`历史数据`按钮，点击跳转或展开历史数据查询

### 13.6 下发历史卡片

> 对应需求：AC-016

- **列表展示**：该实例的配置下发历史记录
- **列表字段**：
  - `下发时间`：格式 `YYYY-MM-DD HH:mm:ss`
  - `下发类型`：创建 / 更新 / 删除
  - `下发状态`：成功 / 失败
  - `网关名称`
- **排序规则**：默认按 `下发时间` 倒序排列
- **分页**：支持分页，`待补充`每页条数

### 13.7 历史数据查询卡片

> 对应需求：AC-020、AC-021

- **点位选择**：
  - 多选下拉选，可选择一个或多个点位
  - 默认选中第一个点位
- **时间范围选择**：
  - 快捷选项：`1小时`、`6小时`、`24小时`、`7天`、`30天`
  - 自定义时间范围：开始时间 + 结束时间
  - 默认：`24小时`
- **展示方式 Tab**：
  - `趋势图`：折线图展示
    - X 轴：时间
    - Y 轴：数值
    - 多点位时不同颜色区分，带图例
    - Hover 显示具体数值和时间
  - `列表`：表格展示
    - 字段：时间、点位名称、数值
    - 按时间倒序排列
- **查询按钮**：点击后根据选择条件查询历史数据
- **数据说明**：
  - 历史数据存储在平台侧时序库
  - `待补充`数据保留时长

## 14. 业务规则与状态反馈

### 14.1 状态规则

- **实例状态三种**：在线 / 离线 / 采集中
  - `在线`：实例关联的网关在线且能通信
  - `离线`：实例关联的网关离线或不能通信
  - `采集中`：实例在线且能采集到点位数值
- **状态实时推送**：状态变化通过 SSE 实时推送到前端

### 14.2 点位归属规则

- 模板级点位：在模板上新增的点位，实例继承后可修改属性，但同步模板版本时会被覆盖
- 设备级点位：在设备实例上新增的点位，同步模板版本时保留不变

### 14.3 网关联联规则

- 一个实例只能分配一个网关，但可以变更网关
- 变更网关后需重新下发配置才能生效

### 14.4 删除规则

- 删除实例需先从网关上删除配置，再删除平台记录
- 网关离线时无法删除配置，实例记录仍保留
- 删除操作不可恢复，需二次确认

### 14.5 下发规则

- 系统比对配置版本号和已下发版本号，有变化则全量下发
- 网关离线时下发失败，提示具体原因

### 14.6 同步模板版本规则

- 同步后，继承点位的修改被模板最新版本覆盖
- 设备级点位保留不变

### 14.7 通用反馈

- **操作成功**：Toast 提示成功信息，3 秒后消失
- **操作失败**：Toast 或弹窗显示具体错误原因
- **加载状态**：按钮置灰 + 文字变化（如「保存中...」）
- **空状态**：列表为空时显示空状态插画 + 文案
- **搜索无结果**：显示「未找到匹配的实例」
- **批量导入错误**：弹窗内显示具体错误项和原因
