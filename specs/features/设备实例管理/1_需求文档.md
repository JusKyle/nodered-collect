# 设备实例管理 · 需求文档

## 1. 功能概述

设备实例管理是系统的核心模块，负责管理具体的设备实例。用户从设备模板创建实例，配置通信参数，分配网关，并管理实例的生命周期（创建、配置、下发、删除）。实例继承模板的点位列表，同时支持个性化定制（修改继承点位、新增设备级点位）。实例详情页提供实时数据展示和历史数据查询能力，为运维人员提供完整的设备监控视图。

## 2. 场景与用户

### 2.1 目标用户

- 工业自动化工程师 — 负责创建设备实例和配置点位
- IoT 系统运维人员 — 负责分配网关、下发配置、监控状态
- 数据采集系统管理员 — 负责批量管理和日常运维

### 2.2 典型场景

**场景一：单个设备接入**
工程师需要接入一台新的 Modbus 温湿度传感器，从对应模板创建实例，填写设备ID和名称，配置 IP 和从站地址，分配到指定网关，点击下发配置，设备开始采集数据。

**场景二：批量设备部署**
管理员需要部署 20 台相同型号的 PLC，准备 Excel 文件包含设备ID、名称、分组、模板ID、网关ID、通信参数，批量导入后一键下发配置，所有设备开始工作。

**场景三：实时数据监控**
运维人员打开某实例详情页，查看点位列表的实时值和最近更新时间，数据通过 SSE 实时推送，无需手动刷新。

**场景四：历史数据分析**
工程师需要分析某温度点位过去一周的变化趋势，在实例详情页选择该点位和时间范围，查看趋势图和列表数据，判断设备运行是否正常。

**场景五：网关变更维护**
某网关需要维护，运维人员将该网关上的实例批量变更到备用网关，重新下发配置，设备继续正常采集。

**场景六：模板版本同步**
模板更新了点位配置，实例列表提示"模板有新版本"，管理员点击同步，实例点位更新为最新版本，重新下发配置生效。

## 3. 核心流程

### 3.1 实例创建流程

**单个创建**
1. 选择设备模板
2. 填写设备ID、名称、分组
3. 实例创建成功，继承模板当前版本的点位列表

**批量导入**
1. 准备 Excel 文件（设备ID、名称、分组、模板ID/名称、网关ID/名称、通信参数）
2. 导入文件
3. 批量导入成功，实例列表新增多条记录

### 3.2 实例配置流程

1. 配置通信参数（IP、端口、从站地址等协议参数）
2. 分配网关（单个或批量）
3. 点位管理：
   - 修改继承点位的属性
   - 新增设备级点位
4. 同步模板版本（当模板有新版本时）

### 3.3 配置下发流程

1. 选择单个或多个实例
2. 点击下发配置
3. 配置下发到对应网关的 Node-RED
4. 下发历史记录新增一条

### 3.4 实例删除流程

1. 点击删除实例
2. 系统下发一次配置到网关，从网关上删除该实例的采集配置
3. 实例记录从平台移除

### 3.5 实时数据展示流程

1. 用户打开实例详情页
2. 点位列表显示实时值 + 最近更新时间
3. 数据通过 SSE 实时推送，自动刷新

### 3.6 历史数据查询流程

1. 选择单个或多个点位
2. 选择时间范围
3. 从时序库查询历史数据
4. 展示列表 + 趋势图

## 4. 边界与异常处理

### 4.1 批量导入

| 异常场景 | 处理方式 |
|---------|---------|
| Excel 中有重复的设备ID | 导入失败，提示重复的设备ID列表 |
| 模板ID不存在 | 导入失败，提示不存在的模板ID |
| 网关ID不存在 | 导入失败，提示不存在的网关ID |

### 4.2 模板版本同步

| 异常场景 | 处理方式 |
|---------|---------|
| 实例有设备级点位 | 设备级点位保留不变 |
| 继承点位被修改过 | 修改会被模板最新版本覆盖 |

### 4.3 配置下发与删除

| 异常场景 | 处理方式 |
|---------|---------|
| 网关离线时下发配置 | 提示"网关离线，下发失败" |
| 网关离线时删除实例 | 提示"网关离线，无法删除配置"，实例记录仍保留 |

### 4.4 状态判定

| 状态 | 判定条件 |
|------|---------|
| 在线 | 网关在线且能通信 |
| 离线 | 网关离线或不能通信 |
| 采集中 | 在线且能采集到点位数值 |

## 5. 业务规则

### 5.1 实例与网关关联规则

- 一个实例只能分配一个网关
- 可以变更网关
- 变更网关后需重新下发配置

### 5.2 点位归属规则

- 在模板上新增的点位 → 模板级
- 在设备上新增的点位 → 设备级
- 继承的点位被实例修改后 → 仍属于模板级，模板更新时会被覆盖

### 5.3 实例删除规则

- 需下发一次配置从网关上删除
- 网关离线时无法删除配置，实例记录保留

### 5.4 实时数据展示规则

- 通过 SSE 推送实时值
- 每个点位显示实时值 + 最近更新时间

### 5.5 历史数据查询规则

- 按点位保存历史数据到时序库
- 可选任意时间范围查询
- 展示列表 + 趋势图
- 支持单个点位查询和多个点位对比查询

## 6. 范围界定

### 6.1 本次做

- 实例创建：单个创建 + 批量导入（Excel）
- 配置通信参数：IP、端口、从站地址等协议参数
- 分配网关：单个分配 + 批量分配
- 点位管理：修改继承点位属性 + 新增设备级点位
- 同步模板版本：当模板有新版本时可更新
- 变更网关：单个变更 + 批量变更
- 下发配置：单个下发 + 批量下发
- 删除实例：下发配置从网关删除 + 移除记录
- 实例详情页：通信参数 + 网关信息 + 采集状态 + 点位实时值 + 下发历史 + 历史数据查询
- 实例列表：名称、模板名称、分组、网关名称、状态
- 列表筛选：按分组 / 模板 / 网关 / 状态筛选
- 状态显示：在线 / 离线 / 采集中

### 6.2 本次不做

- 告警规则配置
- 实例性能监控
- 批量导出实例

## 7. 验收标准（AC）

### 7.1 Happy Path

| 编号 | 验收标准 |
|------|---------|
| AC-001 | Given 用户创建单个设备实例，When 填写设备ID、名称、分组并选择模板，Then 实例创建成功，继承模板当前版本的点位列表。 |
| AC-002 | Given 用户批量导入 Excel，When 文件包含设备ID、名称、分组、模板ID、网关ID、通信参数，Then 批量创建成功，实例列表新增多条记录。 |
| AC-003 | Given 实例已创建，When 用户配置通信参数（IP、端口、从站地址等），Then 配置保存成功。 |
| AC-004 | Given 实例已创建，When 用户分配网关，Then 实例与网关关联。 |
| AC-005 | Given 实例已创建，When 用户批量分配网关（选择多个实例 + 选择网关），Then 所有选中实例与该网关关联。 |
| AC-006 | Given 实例继承了模板点位，When 用户修改某个继承点位的属性，Then 修改保存成功，该点位仍属于模板级。 |
| AC-007 | Given 实例已创建，When 用户新增设备级点位，Then 点位添加成功，属于设备级。 |
| AC-008 | Given 实例绑定的模板有新版本，When 用户点击"同步最新版本"，Then 实例点位更新为模板最新版本。 |
| AC-009 | Given 实例已分配网关，When 用户变更网关，Then 实例与新网关关联，需重新下发配置。 |
| AC-010 | Given 用户点击下发配置，When 选择单个或多个实例，Then 配置下发到对应网关，下发历史记录新增一条。 |
| AC-011 | Given 用户删除实例，When 确认删除，Then 实例记录移除，下发一次配置从网关上删除。 |
| AC-012 | Given 用户在实例列表查看，Then 每一行显示名称、模板名称、分组、网关名称、状态。 |
| AC-013 | Given 用户在实例列表筛选，When 按分组 / 模板 / 网关 / 状态筛选，Then 显示符合条件的实例。 |
| AC-014 | Given 用户点击实例详情，Then 详情页展示通信参数、网关信息、采集状态、点位列表、下发历史。 |
| AC-015 | Given 实例详情页点位列表，When 网关上报点位实时值，Then 每个点位显示实时值 + 最近更新时间（SSE 推送）。 |
| AC-016 | Given 实例详情页下发历史，Then 显示下发时间、下发类型（创建 / 更新 / 删除）、下发状态、网关名称。 |
| AC-017 | Given 实例关联的网关在线且能通信，Then 实例状态为在线。 |
| AC-018 | Given 实例关联的网关离线或不能通信，Then 实例状态为离线。 |
| AC-019 | Given 实例在线且能采集到点位数值，Then 实例状态为采集中。 |
| AC-020 | Given 用户查询点位历史数据，When 选择点位和时间范围，Then 显示该时间范围内的历史数据（列表 + 趋势图）。 |
| AC-021 | Given 用户查询多个点位历史数据，When 选择多个点位和时间范围，Then 显示各点位的历史数据对比（趋势图）。 |

### 7.2 Edge & Error Cases

| 编号 | 验收标准 |
|------|---------|
| AC-022 | Given 用户批量导入 Excel，When 文件中有重复的设备ID，Then 导入失败，提示重复的设备ID列表。 |
| AC-023 | Given 用户批量导入 Excel，When 模板ID不存在，Then 导入失败，提示不存在的模板ID。 |
| AC-024 | Given 用户批量导入 Excel，When 网关ID不存在，Then 导入失败，提示不存在的网关ID。 |
| AC-025 | Given 实例同步到最新版本，When 实例有设备级点位，Then 设备级点位保留不变。 |
| AC-026 | Given 实例同步到最新版本，When 实例的继承点位被修改过，Then 修改会被模板最新版本覆盖。 |
| AC-027 | Given 用户删除实例，When 网关离线，Then 提示"网关离线，无法删除配置"，实例记录仍保留。 |
| AC-028 | Given 用户下发配置，When 网关离线，Then 提示"网关离线，下发失败"。 |

### 7.3 Business Rules

| 编号 | 验收标准 |
|------|---------|
| AC-029 | Given 实例与网关关联，Then 一个实例只能分配一个网关，但可以变更网关。 |
| AC-030 | Given 点位归属判定，Then 在模板上新增的点位属于模板级，在设备上新增的点位属于设备级。 |
| AC-031 | Given 实例删除，Then 需下发一次配置从网关上删除，网关离线时无法删除配置。 |
| AC-032 | Given 点位实时值展示，Then 通过 SSE 推送，每个点位显示实时值 + 最近更新时间。 |
| AC-033 | Given 历史数据查询，Then 按点位保存历史数据，可选任意时间范围，列表 + 趋势图展示，数据存储在平台侧时序库。 |
