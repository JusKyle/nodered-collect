# 设备实例管理 — 功能需求说明书

## 1. 背景与价值

**为什么做**：设备实例是系统运行的核心实体，基于设备模板创建，关联网关，承载实际的数据采集任务。实例管理模块负责实例的完整生命周期管理，包括创建、配置、下发、删除，以及实时数据展示和历史数据查询。

**做成什么样**：用户可以在平台上完成实例的创建、配置、下发，支持批量操作，实时查看采集数据和下发状态，无需登录网关操作。

**关联**：依赖于设备模型管理模块。

---

## 2. 范围

### 本次做

- 实例创建：单个创建 + 批量导入（Excel）
- 配置通信参数：IP、端口、从站地址等协议参数
- 分配网关：单个分配 + 批量分配
- 点位管理：修改继承点位属性 + 新增设备级点位
- 同步模板版本：当模板有新版本时可更新
- 变更网关：单个变更 + 批量变更
- 下发配置：单个下发 + 批量下发
- 删除实例：下发配置从网关删除 + 移除记录
- 实例详情页：通信参数 + 网关信息 + 采集状态 + 点位实时值 + 下发历史 + 历史数据查询
- 实例列表：名称、模板名称、分组、网关名称、状态
- 列表筛选：按分组 / 模板 / 网关 / 状态筛选
- 状态显示：在线 / 离线 / 采集中

### 本次不做

- 告警规则配置
- 实例性能监控
- 批量导出实例

## 4. 需求详述

### 4.1 核心流程

> 这里只描述**主线路径的全局视角**——用户从哪里开始、经过哪些关键节点、到哪里结束。

**单个实例创建流程**
1. 用户选择设备模板
2. 填写设备ID、名称、分组
3. 实例创建成功，继承模板当前版本的点位列表
4. 配置通信参数（IP、端口、从站地址等）
5. 分配网关

**批量导入流程**
1. 用户准备 Excel 文件（设备ID、名称、分组、模板ID、网关ID、通信参数）
2. 导入文件
3. 系统校验：重复ID、模板ID不存在、网关ID不存在
4. 批量创建成功

**下发配置流程**
1. 用户选择实例点击下发
2. 系统比对配置版本号和已下发版本号
3. 有变化则全量下发到网关 Node-RED
4. 下发结果反馈（成功/失败）

**删除实例流程**
1. 用户确认删除实例
2. 系统调用 Node-RED API 删除对应 flow
3. 删除平台实例记录
4. 如网关离线，记录离线状态

### 4.2 交互规则

- 批量导入时如有错误，提示具体错误项和原因
- 删除操作需二次确认弹窗
- 下发失败时显示具体失败原因
- 网关离线时下发和删除操作给予明确提示

### 4.3 业务规则

| 编号 | 规则 |
|------|------|
| BR-001 | 一个实例只能分配一个网关，但可以变更网关 |
| BR-002 | 在模板上新增的点位属于模板级，在设备上新增的点位属于设备级 |
| BR-003 | 删除实例需下发一次配置从网关上删除，网关离线时无法删除配置 |
| BR-004 | 实时值通过 SSE 推送，每个点位显示实时值 + 最近更新时间 |
| BR-005 | 历史数据存在平台侧时序库，可选时间范围查询，支持列表 + 趋势图展示 |

## 5. 验收标准

> AC 是这份文档的核心交付物。后续的技术方案、任务拆分、TDD 测试用例全部从这里派生。

### 5.1 正常流程 (Happy Path)

- [ ] **AC-001**: 单个创建设备实例
  - **Given** 用户创建设备实例
  - **When** 填写设备ID、名称、分组并选择模板
  - **Then** 实例创建成功，继承模板当前版本的点位列表

- [ ] **AC-002**: 批量导入实例
  - **Given** 用户批量导入 Excel
  - **When** 文件包含设备ID、名称、分组、模板ID、网关ID、通信参数
  - **Then** 批量创建成功，实例列表新增多条记录

- [ ] **AC-003**: 配置通信参数
  - **Given** 实例已创建
  - **When** 用户配置通信参数（IP、端口、从站地址等）
  - **Then** 配置保存成功

- [ ] **AC-004**: 分配网关
  - **Given** 实例已创建
  - **When** 用户分配网关
  - **Then** 实例与网关关联

- [ ] **AC-005**: 批量分配网关
  - **Given** 实例已创建
  - **When** 用户选择多个实例 + 选择网关
  - **Then** 所有选中实例与该网关关联

- [ ] **AC-006**: 修改继承点位
  - **Given** 实例继承了模板点位
  - **When** 用户修改某个继承点位的属性
  - **Then** 修改保存成功，该点位仍属于模板级

- [ ] **AC-007**: 新增设备级点位
  - **Given** 实例已创建
  - **When** 用户新增设备级点位
  - **Then** 点位添加成功，属于设备级

- [ ] **AC-008**: 同步模板最新版本
  - **Given** 实例绑定的模板有新版本
  - **When** 用户点击"同步最新版本"
  - **Then** 实例点位更新为模板最新版本

- [ ] **AC-009**: 变更网关
  - **Given** 实例已分配网关
  - **When** 用户变更网关
  - **Then** 实例与新网关关联，需重新下发配置

- [ ] **AC-010**: 下发配置
  - **Given** 用户点击下发配置
  - **When** 选择单个或多个实例
  - **Then** 配置下发到对应网关，下发历史记录新增一条

- [ ] **AC-011**: 删除实例
  - **Given** 用户删除实例
  - **When** 确认删除
  - **Then** 实例记录移除，下发一次配置从网关上删除

- [ ] **AC-012**: 查看实例列表
  - **Given** 用户在实例列表
  - **When** 查看列表
  - **Then** 每行显示名称、模板名称、分组、网关名称、状态

- [ ] **AC-013**: 筛选实例
  - **Given** 用户在实例列表
  - **When** 按分组 / 模板 / 网关 / 状态筛选
  - **Then** 显示符合条件的实例

- [ ] **AC-014**: 查看实例详情
  - **Given** 用户点击实例详情
  - **When** 查看详情页
  - **Then** 详情页展示通信参数、网关信息、采集状态、点位列表、下发历史

- [ ] **AC-015**: 实时值 SSE 推送
  - **Given** 实例详情页点位列表
  - **When** 网关上报点位实时值
  - **Then** 每个点位显示实时值 + 最近更新时间（SSE 推送）

- [ ] **AC-016**: 下发历史展示
  - **Given** 实例详情页下发历史
  - **When** 查看下发记录
  - **Then** 显示下发时间、下发类型（创建 / 更新 / 删除）、下发状态、网关名称

- [ ] **AC-017**: 实例在线状态
  - **Given** 实例关联的网关在线且能通信
  - **When** 状态判定
  - **Then** 实例状态为在线

- [ ] **AC-018**: 实例离线状态
  - **Given** 实例关联的网关离线或不能通信
  - **When** 状态判定
  - **Then** 实例状态为离线

- [ ] **AC-019**: 实例采集中状态
  - **Given** 实例在线且能采集到点位数值
  - **When** 状态判定
  - **Then** 实例状态为采集中

- [ ] **AC-020**: 查询点位历史数据
  - **Given** 用户查询点位历史数据
  - **When** 选择点位和时间范围
  - **Then** 显示该时间范围内的历史数据（列表 + 趋势图）

- [ ] **AC-021**: 查询多个点位历史数据
  - **Given** 用户查询多个点位历史数据
  - **When** 选择多个点位和时间范围
  - **Then** 显示各点位的历史数据对比（趋势图）

### 5.2 边界与异常 (Edge & Error Cases)

- [ ] **AC-022**: 批量导入重复ID
  - **Given** 用户批量导入 Excel
  - **When** 文件中有重复的设备ID
  - **Then** 导入失败，提示重复的设备ID列表

- [ ] **AC-023**: 批量导入模板不存在
  - **Given** 用户批量导入 Excel
  - **When** 模板ID不存在
  - **Then** 导入失败，提示不存在的模板ID

- [ ] **AC-024**: 批量导入网关不存在
  - **Given** 用户批量导入 Excel
  - **When** 网关ID不存在
  - **Then** 导入失败，提示不存在的网关ID

- [ ] **AC-025**: 同步版本保留设备级点位
  - **Given** 实例同步到最新版本
  - **When** 实例有设备级点位
  - **Then** 设备级点位保留不变

- [ ] **AC-026**: 同步版本覆盖修改
  - **Given** 实例同步到最新版本
  - **When** 实例的继承点位被修改过
  - **Then** 修改会被模板最新版本覆盖

- [ ] **AC-027**: 删除实例网关离线
  - **Given** 用户删除实例
  - **When** 网关离线
  - **Then** 提示"网关离线，无法删除配置"，实例记录仍保留

- [ ] **AC-028**: 下发网关离线
  - **Given** 用户下发配置
  - **When** 网关离线
  - **Then** 提示"网关离线，下发失败"

### 5.3 业务规则验证 (Business Rules)

- [ ] **AC-029**: 一个实例一个网关 ← BR-001
  - **Given** 实例与网关关联
  - **Then** 一个实例只能分配一个网关，但可以变更网关

- [ ] **AC-030**: 点位归属判定 ← BR-002
  - **Given** 点位归属判定
  - **Then** 在模板上新增的点位属于模板级，在设备上新增的点位属于设备级

- [ ] **AC-031**: 删除实例清理 ← BR-003
  - **Given** 删除实例
  - **Then** 需下发一次配置从网关上删除，网关离线时无法删除配置

- [ ] **AC-032**: 实时值展示 ← BR-004
  - **Given** 点位实时值展示
  - **Then** 通过 SSE 推送，每个点位显示实时值 + 最近更新时间

- [ ] **AC-033**: 历史数据查询 ← BR-005
  - **Given** 历史数据查询
  - **Then** 按点位保存历史数据，可选时间范围，列表 + 趋势图展示，数据存储在平台侧时序库
