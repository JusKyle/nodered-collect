# 设备模型管理 - 页面功能交互说明

## 1. 页面综述

- **功能描述**：统一管理各类工业设备的采集方案模板，支持模板创建编辑、点位配置、版本管理、批量导入导出和模板复制。
- **页面类型**：后台管理类 (Admin / Table-First)
- **目标用户**：工业自动化工程师、IoT 系统运维人员、数据采集系统管理员
- **核心目标**：用户可以快速定义 Modbus / S7 / OPC UA / MQTT / TCP 等各类设备的采集配置，支持版本管理、批量导入导出，降低多设备管理成本。
- **本阶段范围**：设备模板管理（创建、编辑、版本管理、复制、删除）、点位管理（增删改、批量导入导出 Excel）、协议特殊字段动态切换

---

## 2. 页面结构

- **页面标题**：设备模型
- **页面副标题**：管理设备采集模板和点位配置
- **主体布局**：导航栏 + 顶部筛选区 + 操作按钮区 + 数据表格区
- **主要页面/区域**：
  - 导航栏：采集网关，注册码，设备模型，设备实例，配置记录，系统配置
  - 模板列表页（主页面）
  - 新建设备模板（弹窗）
  - 编辑设备模板（弹窗）
  - 模板详情页（二级页面）
  - 点位管理（详情页内的子模块）
  - 新增/编辑点位（弹窗）
  - 导入点位（弹窗）
  - 历史版本（抽屉）
  - 模板复制确认（弹窗）
  - 删除确认（气泡）

## 3. 模板列表页

### 3.1 搜索/筛选区

- **搜索字段**：
  - `模板名称`：输入框，默认空，模糊搜索
- **筛选字段**：
  - `协议类型`：下拉选，默认`全部`，选项：`全部`、`Modbus TCP`、`Modbus RTU`、`S7`、`OPC UA`、`MQTT`、`TCP`
- **交互**：
  - 输入搜索关键词后需点击`查询`按钮
  - 选择筛选条件后需点击`查询`按钮
  - 搜索与筛选条件叠加生效
  - 点击`重置`清空所有搜索和筛选条件，恢复默认状态

### 3.2 操作按钮区

- **`新建模板`**：主按钮（primary），点击打开`新建设备模板（弹窗）`

### 3.3 列表展示区

- **排序规则**：默认按 `更新时间` 倒序排列。
- **列表字段**：
  - `模板名称`：模板名称，加粗显示，可点击跳转到模板详情页
  - `协议类型`：协议名称，带标签样式
  - `点位数`：当前版本的点位数量
  - `版本号`：当前最新版本号，格式 `vN`
  - `更新时间`：格式 `YYYY-MM-DD HH:mm`
- **行内操作**：
  - `点位`：点击打开点位管理弹窗
  - `编辑`：点击打开编辑设备模板弹窗
  - `复制`：点击打开模板复制确认弹窗
  - `删除`：点击展开删除确认气泡

### 3.4 删除确认（气泡）

- **触发方式**：点击列表行内删除图标按钮
- **气泡位置**：按钮下方，右对齐
- **关闭方式**：点击`取消`、点击气泡外部、删除成功后自动关闭
- **气泡内容**：
  - 标题：「确定删除模板「{模板名称}」吗？」
  - 辅助说明：「删除后无法恢复，请谨慎操作」
  - 操作按钮：
    - `取消`：灰色按钮，关闭气泡
    - `确定`：红色按钮
      - 点击后变为「删除中...」并禁用
      - 调用删除模板 API
      - 成功后刷新列表，关闭气泡
- **受限删除**：
  - 若模板已有实例关联，点击`确定`后提示「该模板已有实例关联，无法删除」
  - 气泡不关闭，保留用户操作上下文

## 4. 新建设备模板（弹窗）

- **打开方式**：点击列表页 `新建模板` 按钮
- **弹窗宽度**：中等（约 560px）
- **弹窗标题**：新建设备模板
- **布局**：仅基本信息表单（点位在详情页配置）
- **底部操作**：
  - `取消`：关闭弹窗，重置表单
  - `保存`：主按钮
    - 点击后按钮变为「保存中...」并禁用
    - 调用创建模板 API
    - 保存成功后版本号为 `v1`
    - 成功后 Toast 提示「创建成功」，关闭弹窗，刷新列表

### 4.1 表单字段

- **`模型ID`**：必填，输入框，最多 50 字符，全局唯一
  - 校验：字母、数字、下划线组合
- **`模型名称`**：必填，输入框，最多 100 字符
- **`协议类型`**：必填，下拉选
  - 选项：`Modbus TCP`、`Modbus RTU`、`S7`、`OPC UA`、`MQTT`、`TCP`
- **`备注`**：选填，文本域，最多 500 字符，3 行高度

## 5. 编辑设备模板（弹窗）

- **打开方式**：点击列表行内 `编辑` 按钮，或模板详情页 `编辑` 按钮
- **弹窗宽度**：中等（约 560px）
- **弹窗标题**：编辑设备模板
- **版本提示**：无版本升级提示（编辑基本信息不升级版本）
- **底部操作**：
  - `取消`：关闭弹窗，不保存修改
  - `保存`：主按钮
    - 点击后按钮变为「保存中...」并禁用
    - 调用更新模板 API
    - 保存成功后版本号**保持不变**
    - 成功后 Toast 提示「保存成功」
    - 关闭弹窗，刷新列表/详情页

### 5.1 表单字段

- 字段与交互同「新建设备模板」
- 协议类型不可修改（灰色禁用）
- 表单回显当前最新版本的基本信息

## 6. 模板详情页（二级页面）

- **入口**：点击列表页模板名称
- **返回**：左上角返回箭头，返回模板列表页

### 6.1 基本信息卡片

- **卡片标题**：基本信息，右上角 `编辑` 按钮
- **布局**：两列键值对
- **字段**：
  - `模型名称`
  - `协议类型`
  - `模型ID`（等宽字体，浅灰背景）
  - `版本号`（版本标签样式）
  - `创建时间`
  - `备注`（跨两列）

### 6.2 点位列表卡片

- **卡片标题**：点位列表，右上角显示点位总数
- **顶部操作**：
  - `导入`：次要按钮，点击打开导入点位弹窗
  - `导出`：次要按钮，点击导出当前模板点位 Excel 文件
  - `新增点位`：主按钮，点击打开新增点位弹窗
- **列表字段**：
  - `点位名称`
  - `地址`（根据协议动态显示：Modbus显示寄存器地址、S7显示DB块号+字节偏移、OPC UA显示节点ID、MQTT显示字段路径、TCP显示解析规则摘要）
  - `数据类型`
  - `读写权限`
  - `单位`
  - `操作`：`编辑`、`删除`
- **空状态**：暂无点位，请点击「新增点位」或「导入点位」添加

## 7. 新增/编辑点位（弹窗）

- **打开方式**：
  - 新增：点击点位列表 `新增点位` 按钮
  - 编辑：点击点位列表行内 `编辑` 按钮
- **弹窗宽度**：大（约 720px）
- **弹窗标题**：
  - 新增：新增点位
  - 编辑：编辑点位
- **副标题**：显示当前协议类型，如「Modbus TCP 协议」
- **底部操作**：
  - `取消`：关闭弹窗，重置表单
  - `确定`：主按钮
    - 点击后按钮变为「保存中...」并禁用
    - 校验通过后保存点位
    - 成功后 Toast 提示「保存成功」，关闭弹窗，刷新点位列表
    - 版本号自动升级（vN → vN+1）

### 7.1 基本信息区

- **分区标题**：基本信息
- **字段**：
  - **`点位ID`**：必填，输入框，最多 50 字符，模板内唯一
    - 校验：字母、数字、下划线组合
  - **`点位名称`**：必填，输入框，最多 100 字符
  - **`描述`**：选填，输入框，最多 200 字符
  - **`数据类型`**：必填，下拉选
    - 选项：`BOOL`、`INT16`、`UINT16`、`INT32`、`UINT32`、`FLOAT32`、`FLOAT64`
  - **`读写权限`**：必填，下拉选，选项：`只读`、`读写`

### 7.2 转换配置区

- **分区标题**：转换配置
- **说明文字**：实际值 = (原始值 × 乘系数) + 偏移量
- **字段**：
  - **`乘系数`**：选填，数字输入框，默认 1
  - **`偏移量`**：选填，数字输入框，默认 0
  - **`精度`**：选填，数字输入框
    - **动态显隐规则**：
      - 当 `数据类型` 为 `FLOAT32` 或 `FLOAT64` 时显示
      - 其他数据类型时隐藏
    - 范围：0-10 的整数
    - 说明：控制浮点类型的小数位数
  - **`单位`**：选填，输入框，最多 20 字符
  - **`采集频率`**：必填，数字输入框，单位 ms，默认 1000，范围 100-86400000
  - **`采集超时`**：选填，数字输入框，单位 ms，默认 3000，范围 100-60000

### 7.3 协议特殊字段区

- **分区标题**：协议特殊字段
- **动态显隐规则**：根据模板的 `协议类型` 动态显示对应点位特殊字段

#### Modbus TCP / Modbus RTU 协议

- **`从站地址`**：必填，数字输入框，范围 1-247
  - 说明：Modbus 从站地址（Unit ID）
- **`功能码`**：必填，下拉选
  - **动态规则**：根据 `读写权限` 动态切换选项
    - 只读时：
      - `01 读线圈`
      - `02 读离散输入`
      - `03 读保持寄存器`
      - `04 读输入寄存器`
    - 读写时（只读选项 + 写入选项）：
      - `03 读保持寄存器`
      - `06 写单个寄存器`
      - `16 写多个寄存器`
      - `04 读输入寄存器`
      - `01 读线圈`
      - `05 写单个线圈`
      - `15 写多个线圈`
      - `02 读离散输入`
- **`寄存器地址`**：必填，数字输入框，范围 0-65535
  - 说明：十进制起始地址

#### S7 协议

- **`DB块号`**：必填，数字输入框，范围 0-65535
  - 说明：数据块编号，如 DB1、DB100
- **`字节偏移`**：必填，数字输入框，单位字节，范围 0-65535
  - 说明：在 DB 块中的起始字节位置
- **`位偏移`**：选填，数字输入框，范围 0-7
  - **动态显隐规则**：
    - 当 `数据类型` 为 `BOOL` 时显示且必填
    - 其他数据类型时隐藏
  - 说明：位地址，0-7

#### OPC UA 协议

- **`命名空间索引`**：必填，数字输入框，默认 0
  - 说明：OPC UA 命名空间索引（Namespace Index）
- **`节点ID`**：必填，输入框
  - 说明：OPC UA 节点标识，如 `ns=2;s=Temperature`

#### MQTT 协议

- **`字段路径 (JSONPath)`**：必填，输入框
  - 说明：用于从订阅报文中提取该点位值，支持 JSONPath，如 `$.data.temperature`

#### TCP 协议

- **`解析规则表达式`**：必填，输入框
  - 说明：用于从设备上报报文中提取该点位值，支持正则表达式，如 `temperature=(\d+\.?\d*)`

### 7.4 校验规则

- 实时校验（输入时）+ 提交时全量校验
- 错误提示：输入框红框 + 下方红色错误文字
- 协议特殊字段必填项未填写时，保存失败，提示「协议必填字段未填写」

## 8. 导入点位（弹窗）

- **打开方式**：点击点位列表 `导入` 按钮
- **弹窗宽度**：中等（约 480px）
- **弹窗标题**：导入点位

### 8.1 弹窗内容

- **模板下载**：
  - 文字说明：「请先下载模板，按模板格式填写后上传」
  - `下载模板`：文字链接按钮，点击下载 Excel 模板文件
- **文件上传区**：
  - 支持拖拽上传或点击选择文件
  - 支持格式：`.xlsx`、`.xls`
  - 文件大小限制：10MB
  - 已选文件显示文件名和大小
- **说明文字**：
  - 导入后将添加到当前点位列表
  - 如有点位 ID 重复，导入失败并提示重复列表

### 8.2 底部操作

- `取消`：关闭弹窗
- `导入`：主按钮
  - 未选择文件时禁用
  - 点击后按钮变为「导入中...」并禁用
  - 调用导入 API
  - 导入成功：
    - Toast 提示「导入成功，共导入 N 个点位」
    - 关闭弹窗，刷新点位列表
    - 版本号自动升级
  - 导入失败：
    - 弹窗内显示错误信息
    - 若存在重复点位 ID，提示「导入失败，存在重复点位ID：ID1、ID2...」

## 9. 导出点位

- **触发方式**：点击点位列表 `导出` 按钮
- **交互**：
  - 点击后按钮变为「导出中...」并禁用
  - 调用导出 API，生成 Excel 文件

  - 文件名格式：`{模板名称}_点位_{版本号}_{YYYYMMDD}.xlsx`
  - 按钮恢复可用状态
- **导出内容**：
  - 包含当前版本的所有点位信息
  - 包含公共字段和协议特殊字段

## 12. 业务规则与状态反馈

### 12.1 版本管理规则

- 编辑基本信息**不**产生新版本
- 修改点位（新增、编辑、删除、导入）自动产生新版本
- 版本号从 `v1` 开始数字递增（v1 → v2 → v3...）
- 旧版本保留，可通过历史版本查看
- 复制模板产生的新模板版本号为 `v1`

### 12.2 值转换规则

- 实际值 = (原始值 × 乘系数) + 偏移量
- 精度字段仅对 FLOAT32/FLOAT64 生效，控制小数位数

### 12.3 协议字段显隐规则

- 点位精度字段仅在数据类型为 FLOAT32/FLOAT64 时显示
- S7 协议的位偏移字段仅在数据类型为 BOOL 时显示
- Modbus 功能码选项根据读写权限动态切换
  - 只读：4 个功能码（01/02/03/04）
  - 读写：8 个功能码（增加 05/06/15/16）

### 12.4 删除限制规则

- 已有实例关联的模板无法删除
- 删除时提示「该模板已有实例关联，无法删除」

### 12.5 导入校验规则

- 导入点位时检查点位 ID 重复性
- 存在重复时导入失败，提示重复的点位 ID 列表

### 12.6 通用反馈

- **操作成功**：Toast 提示成功信息，3 秒后消失
- **操作失败**：Toast 或弹窗显示具体错误原因
- **加载状态**：按钮置灰 + 文字变化（如「保存中...」）
- **空状态**：列表为空时显示空状态插画 + 文案
- **搜索无结果**：显示「未找到匹配的模板」
