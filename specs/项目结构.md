### 2.1 后端目录说明

| 目录/文件 | 职责说明 |
|-----------|---------|
| `prisma/` | Prisma ORM 数据模型定义和迁移文件 |

| `src/modules/` | 业务模块（网关、设备模型、设备实例、同步、注册） |
| `src/services/` | 核心服务（心跳、下发引擎、MQTT、Token） |

| `src/utils/` | 工具函数（日志、响应、加密） |
| `src/types/` | 全局 TypeScript 类型定义 |

## 6. 核心模块映射关系

### 6.1 业务模块与目录映射

| 业务模块 | 后端目录 | 前端目录 | 插件目录 |
|---------|---------|---------|---------|
| 采集网关管理 | `backend/src/modules/gateway/` | `frontend/src/pages/gateway/` | - |
| 设备模型管理 | `backend/src/modules/device-model/` | `frontend/src/pages/device-model/` | - |
| 设备实例管理 | `backend/src/modules/device-instance/` | `frontend/src/pages/device-instance/` | - |
| 配置下发与同步 | `backend/src/modules/sync/` | `frontend/src/pages/sync/` | `nodered-plugin/src/services/config-sync.service.ts` |
| 网关注册 | `backend/src/modules/registration/` | `frontend/src/pages/gateway/RegistrationCodeModal.tsx` | `nodered-plugin/src/services/registration.service.ts` |
| 心跳检测 | `backend/src/services/heartbeat.service.ts` | - | `nodered-plugin/src/services/heartbeat.service.ts` |
| 数据缓存补发 | - | `frontend/src/pages/sync/CacheRepairTab.tsx` | `nodered-plugin/src/services/data-cache.service.ts` |

### 6.2 数据流与目录关系

```
前端页面 (frontend/src/pages/)
    │
    ├──→ API 请求 (frontend/src/api/)
    │       │
    │       └──→ 后端 Controller (backend/src/modules/*/controller.ts)
    │               │
    │               ├──→ Service (backend/src/modules/*/service.ts)
    │               │       │
    │               │       ├──→ Repository (backend/src/modules/*/repository.ts)
    │               │       │       │
    │               │       │       └──→ Prisma Schema (backend/prisma/schema.prisma)
    │               │       │
    │               │       └──→ 下发引擎 (backend/src/services/dispatch.service.ts)
    │               │               │
    │               │               └──→ Node-RED API (HTTP)
    │               │
    │               └──→ MQTT Broker (backend/src/services/mqtt.service.ts)
    │                       │
    │                       └──→ Node-RED 插件 (nodered-plugin/src/services/)
    │                               │
    │                               └──→ SQLite 存储 (nodered-plugin/src/store/sqlite-store.ts)
```

## 7. 关键文件说明

### 7.1 后端关键文件

| 文件 | 说明 |
|------|------|
| `backend/src/main.ts` | Express 应用入口，启动服务器 |
| `backend/src/app.ts` | Express 应用配置，注册中间件和路由 |
| `backend/prisma/schema.prisma` | 数据库模型定义（网关、设备模型、实例、点位、同步记录等） |
| `backend/src/services/dispatch.service.ts` | 配置下发引擎，调用 Node-RED API |
| `backend/src/services/heartbeat.service.ts` | 心跳检测服务，管理网关在线状态 |

### 7.2 前端关键文件

| 文件 | 说明 |
|------|------|
| `frontend/src/main.tsx` | React 应用入口 |
| `frontend/src/App.tsx` | 根组件，路由配置 |
| `frontend/src/layouts/MainLayout.tsx` | 主布局（侧边栏 + 头部 + 内容区） |
| `frontend/src/api/axios.ts` | Axios 实例配置（拦截器、baseURL） |

### 7.3 插件关键文件

| 文件 | 说明 |
|------|------|
| `nodered-plugin/src/index.ts` | Node-RED 插件注册入口 |
| `nodered-plugin/src/nodes/device-manager/` | 配置节点，存储设备实例元数据 |
| `nodered-plugin/src/services/registration.service.ts` | 注册服务，使用注册码绑定平台 |
| `nodered-plugin/src/services/data-cache.service.ts` | 数据缓存服务，断网时存储采集数据 |

## 8. 扩展性设计

### 8.1 预留扩展目录

| 目录 | 用途 |
|------|------|
| `backend/src/modules/user/` | 用户权限管理（RBAC） |

- 使用 `.env` 文件管理环境变量
- 配置集中在 `backend/src/config/` 目录
- 支持开发、测试、生产多环境配置
