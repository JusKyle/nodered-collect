# 设备实例管理 — 产品需求说明书

## 1. 背景与价值

**为什么做**：设备模型定义了"设备类型"，但实际接入时需要为每一台具体的设备创建"设备实例"。每台设备有独立的地址、独立的网关绑定、独立的运行状态。设备实例管理模块将模型转化为实际可运行的设备，是连接"模型定义"和"实际采集"的桥梁。

**做成什么样**：从设备模型快速创建设备实例（支持批量），实例自动继承模型点位并可添加自定义点位。支持绑定网关、下发配置、同步模型变更，实时显示每个实例的运行状态。

**关联**：产品概述 §3.2 模块三、§5 配置下发方案、§9.3 验收标准

---

## 2. 范围

### 本次做

- 设备实例列表展示（名称、模型、网关、地址、状态、点位数量）
- 从设备模型创建设备实例（单个创建）
- 批量创建设备实例（同一模型多设备）
- 实例基本信息编辑
- 添加自定义点位
- 更改绑定的网关
- 下发配置到网关（状态：待同步 → 运行中）
- 批量下发配置
- 同步模型点位（模型变更后手动或自动同步到实例）
- 两步删除（先解除下发 → 再删除）
- 查看实例完整点位列表
- 列表搜索（按名称）、筛选（按模型/网关/状态）
- 四态状态管理（未绑定、待同步、运行中、离线）

### 本次不做

- 设备实例运行数据查看（数据采集展示在后续模块）
- 设备实例分组/标签管理
- 设备实例迁移（从一个网关迁移到另一个网关时自动处理配置变更）

## 4. 需求详述

### 4.1 核心流程

**主流程 A — 单个创建设备实例**

1. 运维人员在「设备实例」页面点击 `新建实例` 按钮，打开「新建设备实例」弹窗
2. 选择设备模型（从启用状态的模型列表中选择）
3. 选择绑定网关（从已注册的网关列表中选择）
4. 填写设备地址（IP/COM口号）
5. 填写实例名称
6. 点击 `保存`，实例创建成功，状态为"待同步"（如已选网关）或"未绑定"（如未选网关）

**主流程 B — 批量创建设备实例**

1. 运维人员点击 `批量新建` 按钮，打开「批量新建设备实例」弹窗
2. 选择设备模型
3. 选择绑定网关（可选择"暂不绑定"）
4. 填写多个设备地址和名称（每行一个，支持批量输入框）
5. 点击 `保存`，同时创建多个设备实例

**主流程 C — 下发配置**

1. 运维人员在列表中找到状态为"待同步"的实例
2. 点击 `下发配置`
3. 系统将实例配置推送到绑定的网关
4. 下发成功后，实例状态变为"运行中"

**主流程 D — 同步模型点位**

1. 设备模型的点位发生变更（新增/修改/删除）
2. 所有关联实例自动同步点位变更
3. 实例的点位列表更新

### 4.2 交互规则

- **删除操作**：两步删除，弹出确认气泡"确定解除该实例的下发配置吗？解除后实例状态变为'待同步'。" → 解除后再次点击"删除" → 弹出二次确认气泡"确定删除实例「XXX」吗？"
- **下发操作**：点击 `下发配置` 后按钮置灰，显示"下发中..."；成功后绿色提示"下发成功"，状态变为"运行中"；失败时红色提示失败原因
- **必填校验**：表单提交时实时校验必填项，错误字段红框 + 下方红字提示
- **重复点击防抖**：保存/下发按钮点击后置灰，请求返回后恢复
- **状态颜色**：未绑定（灰色标签）、待同步（黄色标签）、运行中（绿色标签）、离线（红色标签）
- **列表自动刷新**：实例状态每 5 秒自动刷新

### 4.3 业务规则

| 编号 | 规则 |
|------|------|
| BR-001 | 实例从模型继承的点位与模型保持关联，模型变更后自动同步到实例 |
| BR-002 | 实例只能添加自定义点位，不能修改或删除继承的点位 |
| BR-003 | 设备实例状态：未绑定 → 待同步 → 运行中 → 离线，运行中 ↔ 离线可自动切换 |
| BR-004 | 网关离线时，该网关上的所有设备实例状态自动变为"离线"；网关恢复后自动变为"运行中" |
| BR-005 | 已下发的设备实例删除需两步：先解除下发 → 再删除 |
| BR-006 | 只有启用状态的设备模型可用于创建设备实例 |
| BR-007 | 支持批量创建、批量下发配置 |
| BR-008 | 同一实例内点位编码唯一（包括继承和自定义点位） |

## 5. 功能需求

### 5.1 设备实例管理 - 实例列表页

#### 页面综述
- **功能描述**：管理所有设备实例，支持创建、编辑、下发、同步、删除等操作
- **权限控制**：运维人员、设备管理人员可查看；仅运维人员可执行创建、编辑、下发、删除

#### 筛选/搜索区
- **搜索字段**：
  - `实例名称`：输入框，模糊匹配，默认空
  - `所属模型`：下拉选，选项 `全部` + 所有模型名称，默认 `全部`
  - `绑定网关`：下拉选，选项 `全部` + 所有网关名称，默认 `全部`
  - `状态`：下拉选，选项 `全部` / `未绑定` / `待同步` / `运行中` / `离线`，默认 `全部`
- **交互**：
  - 点击 `查询`：更新列表数据，重置页码为 1
  - 点击 `重置`：清空所有筛选条件
  - 切换下拉选：自动触发查询

#### 列表展示区
- **排序规则**：默认按 `创建时间` 倒序排列
- **列表字段**：
  - `实例名称`：文本，最多展示 1 行，超长省略号
  - `所属模型`：文本，显示模型名称
  - `绑定网关`：文本，显示网关名称；未绑定时显示"未绑定"
  - `设备地址`：文本，如"192.168.1.100"或"COM3"
  - `状态`：标签，灰色=未绑定，黄色=待同步，绿色=运行中，红色=离线
  - `点位数量`：文本，格式"继承 X / 自定义 Y"
  - `创建时间`：文本，格式"YYYY-MM-DD HH:mm"
  - `最近修改时间`：文本，格式"YYYY-MM-DD HH:mm"
- **批量操作**：
  - `批量下发`：勾选多个"待同步"状态的实例后显示，点击弹出确认气泡
  - `批量解除下发`：勾选多个"运行中"或"离线"状态的实例后显示

#### 操作列
- `编辑`：打开 `编辑设备实例（弹窗）`
- `更改网关`：打开 `更改网关（弹窗）`
- `下发配置`：下发实例配置到网关（仅"待同步"状态显示）
- `解除下发`：停止网关侧采集（仅"运行中"/"离线"状态显示）
- `同步模型点位`：手动触发从模型同步最新点位（用于自动同步失败场景）
- `查看点位`：打开 `查看点位（弹窗）`
- `删除`：两步删除操作

#### 功能按钮
- `新建实例`：打开 `新建设备实例（弹窗）`
- `批量新建`：打开 `批量新建设备实例（弹窗）`

### 5.2 新建设备实例（弹窗）

#### 打开方式
点击列表页 `新建实例` 按钮

#### 表单字段
- `所属模型`：必填，下拉选，仅显示启用状态的模型；选择后自动显示模型的点位数量
- `绑定网关`：选填，下拉选，显示所有网关名称；可选"暂不绑定"
- `设备地址`：必填，1-100 字符
- `实例名称`：必填，1-50 字符
- `描述`：选填，最多 200 字符

#### 点位预览区
- 选择模型后，自动显示该模型的点位列表预览（只读）
- 显示：点位名称、编码、数据类型、寄存器地址、采集频率

#### 按钮
- `取消`：关闭弹窗
- `保存`：创建实例

### 5.3 批量新建设备实例（弹窗）

#### 打开方式
点击列表页 `批量新建` 按钮

#### 表单字段
- `所属模型`：必填，下拉选（同单个创建）
- `绑定网关`：选填，下拉选（同单个创建）
- `设备列表`：多行输入框，每行格式为"设备地址,实例名称"；也可只填地址，名称自动生成
- `命名规则`：选填，可设置自动生成名称的前缀和起始编号（如"PLC-{001}"）

#### 预览区
- 解析输入的设备列表，显示预览表格
- 每行显示：序号、设备地址、实例名称、状态（可创建/已存在/格式错误）

#### 按钮
- `取消`：关闭弹窗
- `保存`：批量创建所有可创建的实例

### 5.4 编辑设备实例（弹窗）

#### 打开方式
点击列表行 `编辑` 按钮

#### 基本信息区
- `实例名称`：可修改
- `设备地址`：可修改
- `描述`：可修改
- `所属模型`：只读，不可修改
- `绑定网关`：只读（通过 `更改网关` 操作修改）

#### 自定义点位区
- 列表显示现有自定义点位
- `添加自定义点位` 按钮：在列表末尾新增一行，字段同模型点位
- 自定义点位可修改、可删除
- 继承的点位在此区域只读显示（不可修改/删除），用"继承"标签标识

#### 按钮
- `取消`：关闭弹窗
- `保存`：保存修改

### 5.5 更改网关（弹窗）

#### 打开方式
点击列表行 `更改网关` 按钮

#### 表单字段
- `当前网关`：只读，显示当前绑定的网关名称
- `新网关`：必填，下拉选，从已注册网关中选择

#### 提示
- 顶部黄色提示条："更改网关后，如需在新网关上运行，请手动执行'下发配置'。"

#### 按钮
- `取消`：关闭弹窗
- `保存`：更新绑定网关

### 5.6 下发配置 / 解除下发（气泡）

#### 打开方式
点击列表行 `下发配置` 或 `解除下发` 按钮

#### 下发配置提示文案
```
确定下发配置到网关「XXX」吗？
[取消] [确定]
```

#### 解除下发提示文案
```
确定解除该实例的下发配置吗？
解除后实例状态变为"待同步"。
[取消] [确定]
```

#### 批量下发提示文案
```
确定批量下发选中的 X 个实例配置吗？
[取消] [确定]
```

### 5.7 同步模型点位（气泡）

#### 打开方式
点击列表行 `同步模型点位` 按钮

#### 提示文案
```
将从模型「XXX」同步最新点位配置。
继承点位将更新为模型最新版本，自定义点位不受影响。
[取消] [确定]
```

### 5.8 查看点位（弹窗）

#### 打开方式
点击列表行 `查看点位` 按钮

#### 列表展示
- 字段：`点位名称`、`点位编码`、`数据类型`、`读写权限`、`寄存器地址`、`采集频率(ms)`、`单位`、`来源`（标签：继承 / 自定义）
- 支持按"来源"筛选切换查看：全部 / 仅继承 / 仅自定义
- 排序：按点位编码正序排列

### 5.9 删除确认（气泡）

#### 打开方式
点击列表行 `删除` 按钮

#### 第一步提示（已下发的实例）
```
该实例当前处于"运行中"状态，需先解除下发。
[取消] [解除下发]
```

#### 第二步提示
```
确定删除实例「XXX」吗？
删除后不可恢复。
[取消] [确定]
```

## 6. 验收标准

### 6.1 正常流程 (Happy Path)

- [ ] **AC-001**: 单个创建设备实例 - 选择模型和网关
  - **Given** 运维人员在「新建设备实例」弹窗中选择了模型、网关，填写了设备地址和名称
  - **When** 点击 `保存`
  - **Then** 弹窗关闭，设备实例列表新增一行，状态显示为"待同步"

- [ ] **AC-002**: 单个创建设备实例 - 暂不绑定网关
  - **Given** 运维人员选择了模型和设备地址，但未选择网关
  - **When** 点击 `保存`
  - **Then** 实例创建成功，状态显示为"未绑定"

- [ ] **AC-003**: 批量创建设备实例
  - **Given** 运维人员在「批量新建」弹窗中选择了模型和网关，填写了 3 个设备地址和名称
  - **When** 点击 `保存`
  - **Then** 列表新增 3 个实例，状态均为"待同步"

- [ ] **AC-004**: 批量创建 - 自动命名
  - **Given** 运维人员设置命名规则为"PLC-{001}"，填写了 2 个设备地址
  - **When** 点击 `保存`
  - **Then** 实例名称自动生成为"PLC-001"和"PLC-002"

- [ ] **AC-005**: 下发配置 - 单个实例
  - **Given** 实例状态为"待同步"，已绑定网关
  - **When** 点击 `下发配置`，在确认气泡中点击 `确定`
  - **Then** 实例状态变为"运行中"，顶部显示绿色"下发成功"提示

- [ ] **AC-006**: 下发配置 - 批量实例
  - **Given** 列表中有 3 个"待同步"状态的实例
  - **When** 勾选这 3 个实例后点击 `批量下发`
  - **Then** 3 个实例状态均变为"运行中"

- [ ] **AC-007**: 添加自定义点位
  - **Given** 实例已创建，有 5 个继承点位
  - **When** 运维人员在「编辑设备实例」弹窗中添加 1 个自定义点位后点击 `保存`
  - **Then** 列表中该实例的点位数量显示为"继承 5 / 自定义 1"

- [ ] **AC-008**: 查看实例点位
  - **Given** 实例有 5 个继承点位和 1 个自定义点位
  - **When** 点击 `查看点位`
  - **Then** 弹窗中显示 6 个点位，其中 5 个标签为"继承"，1 个标签为"自定义"

- [ ] **AC-009**: 按模型筛选
  - **Given** 列表中有属于模型 A 和模型 B 的实例
  - **When** 运维人员将模型筛选切换为"模型 A"
  - **Then** 列表仅显示属于模型 A 的实例

- [ ] **AC-010**: 按状态筛选
  - **Given** 列表中有"运行中"和"待同步"的实例
  - **When** 运维人员将状态筛选切换为"运行中"
  - **Then** 列表仅显示"运行中"状态的实例

- [ ] **AC-011**: 同步模型点位
  - **Given** 模型已修改点位，实例继承的点位需要更新
  - **When** 运维人员点击 `同步模型点位`
  - **Then** 实例的继承点位更新为模型最新版本

- [ ] **AC-012**: 更改网关
  - **Given** 实例当前绑定网关 A
  - **When** 运维人员点击 `更改网关`，选择网关 B 后保存
  - **Then** 实例的绑定网关变为网关 B，绑定网关节段更新

- [ ] **AC-013**: 解除下发
  - **Given** 实例状态为"运行中"
  - **When** 点击 `解除下发`，确认后
  - **Then** 实例状态变为"待同步"

- [ ] **AC-014**: 删除实例 - 待同步状态
  - **Given** 实例状态为"待同步"（未下发）
  - **When** 运维人员点击 `删除`，二次确认后
  - **Then** 实例从列表中消失

- [ ] **AC-015**: 模型变更自动同步
  - **Given** 模型 A 的点位配置发生变更（新增 1 个点位）
  - **When** 查看使用模型 A 的实例点位
  - **Then** 实例的继承点位自动包含新点位

### 6.2 边界与异常 (Edge & Error Cases)

- [ ] **AC-016**: 表单必填项缺失
  - **Given** 运维人员点击 `保存` 创建实例
  - **When** 未选择模型或未填写设备地址
  - **Then** 对应字段红框高亮，下方红字提示，不关闭弹窗

- [ ] **AC-017**: 设备地址格式错误
  - **Given** 运维人员填写了无效的 IP 地址或 COM 口号
  - **When** 失焦或点击 `保存`
  - **Then** 设备地址字段红框，下方红字提示"请输入有效的设备地址"

- [ ] **AC-018**: 自定义点位编码与继承点位冲突
  - **Given** 继承点位中已存在编码"temp_001"
  - **When** 运维人员添加相同编码的自定义点位
  - **Then** 编码字段红框提示"该编码已存在"

- [ ] **AC-019**: 下发配置失败 - 网关离线
  - **Given** 实例绑定的网关当前状态为"离线"
  - **When** 点击 `下发配置`
  - **Then** 顶部红色提示"网关离线，无法下发配置"，实例状态保持"待同步"

- [ ] **AC-020**: 删除已下发的实例 - 两步操作
  - **Given** 实例状态为"运行中"
  - **When** 运维人员点击 `删除`
  - **Then** 气泡提示需先解除下发，点击"解除下发"后实例状态变为"待同步"，再次点击"删除"才能删除

- [ ] **AC-021**: 网关离线 - 实例状态联动
  - **Given** 实例状态为"运行中"，绑定网关 A
  - **When** 网关 A 变为离线
  - **Then** 该实例状态自动变为"离线"

- [ ] **AC-022**: 网关恢复 - 实例状态自动恢复
  - **Given** 网关从离线恢复为在线
  - **When** 查看绑定该网关的实例
  - **Then** 实例状态从"离线"自动恢复为"运行中"

- [ ] **AC-023**: 批量创建 - 部分失败

  - **When** 点击 `保存`
  - **Then** 2 个合法的实例创建成功，1 个失败，页面显示红色提示"X 个创建失败"并列出失败原因

- [ ] **AC-024**: 仅启用状态的模型可选择

  - **When** 运维人员在「新建设备实例」弹窗中查看模型下拉列表
  - **Then** 模型 B 不显示或不可选（置灰）

  - **When** 查看表单字段
  - **Then** "所属模型"字段为只读，不可修改

- [ ] **AC-026**: BR-001 继承点位与模型关联同步

  - **Given** 实例状态为"待同步"
