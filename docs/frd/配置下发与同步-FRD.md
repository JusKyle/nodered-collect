# 配置下发与同步

平台通过 HTTP API 将设备配置下发到 Node-RED 网关，网关自动生成设备节点开始采集。采集数据通过 MQTT 上报到平台。断网期间数据本地 SQLite 缓存，网络恢复后自动补发。支持云端统一下发和 Node-RED 本地微调双向同步。

## 通用交互与约束

- 权限控制：运维人员、设备管理人员可查看
- 列表：默认按 `下发时间` 倒序排列
- 下发中状态：点击 `下发配置` 后，按钮置灰显示"下发中..."，列表状态变为"下发中"
- 下发失败提示：下发失败后，顶部红色提示失败原因，状态保持"待同步"
- 补发进度展示：网络恢复后，网关详情页显示"正在补发 X 条数据"，进度条实时更新
- 日志查看：点击 `查看日志` 打开下发日志弹窗，显示时间、操作、结果
- 本地微调回传：Node-RED 中修改设备节点后，平台侧设备实例自动更新，显示"本地微调"标记

## 【配置下发管理】产品页面功能描述及交互描述

### 说明

- 配置下发使用 HTTP API 调用 Node-RED `/admin/flows`
- 采集数据通过 MQTT 上报到平台
- 下发失败后自动重试最多 3 次，间隔递增（10s → 30s → 60s）
- 断网期间采集数据写入本地 SQLite 数据库缓存
- 网络恢复后自动补发缓存数据，按时间顺序，带原始时间戳
- 缓存数据补发成功后自动删除
- 设备实例修改后自动重新下发配置到网关
- Node-RED 中本地微调的配置可回传到平台
- Node-RED 中可手动选择绑定平台已下发的设备实例

### 下发日志页面

- 搜索条件：
  - `网关名称`：下拉选，选项 `全部` + 所有网关名称，默认 `全部`
  - `设备实例`：下拉选，选项 `全部` + 所有实例名称，默认 `全部`
  - `结果`：下拉选，选项 `全部` / `成功` / `失败`，默认 `全部`
  - `时间范围`：日期选择器，默认近 7 天
- 按钮：
  - `查询`：更新列表数据，重置页码为 1
  - `重置`：清空所有筛选条件
- 列表展示：
  - 字段：`下发时间`（文本，格式"YYYY-MM-DD HH:mm:ss"）、`目标网关`（文本，显示网关名称）、`设备实例`（文本，显示实例名称）、`操作类型`（标签，`下发` / `重新下发` / `解除下发`）、`结果`（标签，绿色=成功，红色=失败）、`重试次数`（数字，显示重试了几次）、`备注`（文本，显示失败原因或成功提示）
- 行内操作：
  - `查看详情`：打开 `下发详情（弹窗）`

### 下发详情（弹窗）

- 打开方式：点击日志列表行 `查看详情` 按钮
- 展示内容：
  - `下发时间`：精确到秒
  - `目标网关`：网关名称 + 地址:端口
  - `设备实例`：实例名称 + 设备地址
  - `操作类型`：下发/重新下发/解除下发
  - `下发内容`：JSON 格式，可折叠/展开
  - `返回结果`：JSON 格式，可折叠/展开
  - `重试记录`：时间、结果、原因
  - `最终结果`：成功/失败

### 网关详情页 - 数据同步面板

- 位置：网关详情页右侧或下方
- 同步状态区：
  - `网关状态`：在线/离线标签
  - `数据上报状态`：正常/异常标签
  - `MQTT 连接状态`：已连接/未连接标签
  - `最近上报时间`：文本，格式"YYYY-MM-DD HH:mm:ss"
  - `今日上报条数`：数字
- 缓存与补发区：
  - `缓存数据条数`：数字（仅离线时有值）
  - `补发进度`：进度条 + 文本"正在补发 X/Y 条"（仅恢复网络时有值）
  - `补发状态`：未开始/补发中/已完成/部分失败

### Node-RED 设备节点（混合配置模式）

#### 配置节点（Device Mgr）
- 存储设备实例元数据（设备 ID、名称、协议、连接信息）
- 运行时提供配置信息给采集节点

#### 设备实例节点
- 接收配置，实例化点位，管理采集生命周期
- 支持手动选择绑定平台已下发的设备实例

#### 协议采集节点
- 按协议采集数据（复用现有节点如 node-red-contrib-modbus）
- 支持参数微调（采集频率、超时时间等）
- 微调后自动回传到平台

#### 数据处理节点
- 单位转换、阈值检测、数据格式化
- 支持配置转换公式

#### MQTT Output 节点
- 将采集数据发送到平台 MQTT Broker
- 数据格式包含：设备 ID、点位编码、值、时间戳、质量标志

### Node-RED 手动绑定设备实例（弹窗）

- 打开方式：在 Node-RED 中添加设备实例节点时
- 展示内容：
  - 下拉列表展示平台已下发到该网关的设备实例
  - 每个选项显示：设备实例名称、所属模型、设备地址、点位数量
- 操作：
  - 选择设备实例后自动带入点位配置
  - 支持后期解绑和重新绑定
