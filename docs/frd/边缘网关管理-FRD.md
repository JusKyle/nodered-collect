# 边缘网关管理

管理所有接入平台的 Node-RED 网关，支持注册、编辑、删除、测试连接、查看已下发设备，实时显示每个网关的在线状态、版本信息、运行状态和设备数量。

## 通用交互与约束

- 权限控制：运维人员、设备管理人员可查看；仅运维人员可执行编辑、删除、新建
- 列表：默认按 `创建时间` 倒序排列；状态、最后在线时间、运行状态、已下发/运行中数量每 5s 自动刷新，提供 `暂停自动刷新` / `恢复自动刷新` 切换按钮
- 新建/编辑：必填项缺失时字段下方红字提示"请输入 XXX"；端口超出范围时红字提示"端口号需在 1-65535 之间"；地址格式错误时红字提示"请输入合法的 IP 或域名"；保存按钮点击后置灰，请求返回后恢复
- 删除：弹出二次确认气泡，文字"确定删除网关「XXX」吗？该网关下 N 个设备将转为未绑定状态"；删除后其下已下发设备的网关绑定关系解除，设备状态变更为"未绑定"，但设备本身不删除
- 状态切换：心跳方向为「网关 → 平台推送」，间隔 30s；平台连续 3 次未收到心跳（90s）则判定为离线；平台收到下次心跳即自动恢复为在线，无需人工操作
- Token 失效：Admin Token 失效时，下发/心跳/测试连接操作均返回 401，平台需明显提示用户重新填写

## 【边缘网关管理】产品页面功能描述及交互描述

### 说明

- 网关名称允许重名，平台以内部 ID 唯一标识
- 注册码有效期 10 分钟，过期失效需重新生成
- 端口号范围 1-65535
- 离线网关在列表中行底色变浅灰，状态点为灰色
- Admin Token 输入框默认脱敏（密码模式），提供"显示/隐藏"切换按钮

### 网关列表页

- 搜索条件：
  - `网关名称`：输入框，模糊匹配，默认空
  - `状态`：下拉选，选项 `全部` / `在线` / `离线`，默认 `全部`
- 按钮：
  - `查询`：更新列表数据，重置页码为 1
  - `重置`：清空所有筛选条件
  - `新建网关`：打开 `新建网关（弹窗）`
  - `生成注册码`：打开 `生成注册码（弹窗）`
- 列表展示：
  - 字段：`网关名称`（文本，最多展示 1 行，超长省略号）、`状态`（状态点 + 文字，绿色=在线，灰色=离线）、`最后在线时间`（文本，离线时显示"最后在线：YYYY-MM-DD HH:mm:ss"）、`地址：端口`（文本，IP:Port 格式）、`设备数量`（文本，格式"已下发 X / 运行中 Y"）、`Node-RED 版本`（文本，如 "v3.1.0"，离线时显示"-"）、`运行状态`（标签，正常/异常，离线时显示"离线")
- 行内操作：
  - `测试连接`：调用 Node-RED API 验证 Token，结果用 Message 提示（成功绿色"连接成功"，失败红色显示原因）
  - `编辑`：打开 `编辑网关（弹窗）`，回显当前数据
  - `查看已下发设备`：跳转至该网关的设备实例列表页
  - `删除`：弹出二次确认气泡，确认后删除

### 新建网关（弹窗）

- 打开方式：点击列表页 `新建网关` 按钮
- 按钮：
  - `测试连接`：调用 Node-RED API 验证 Token + 网络连通性
  - `取消`：关闭弹窗
  - `确认`：保存网关
- 表单字段：
  - `网关名称`：必填，1-50 字符，允许重名
  - `地址`：必填，IP 或域名格式校验
  - `端口`：必填，整数 1-65535，默认 1880
  - `Admin Token`：必填，密码模式输入，提供"显示/隐藏"按钮
  - `描述`：选填，最多 200 字符
- 校验规则：
  - 必填项缺失：字段下方红字提示"请输入 XXX"
  - 端口超出范围：红字提示"端口号需在 1-65535 之间"
  - 地址格式错误：红字提示"请输入合法的 IP 或域名"
- 行为：
  - 测试连接失败：弹窗中红色提示失败原因（如"Token 无效"、"无法连接到目标地址"），不关闭弹窗
  - 测试连接成功：弹窗下方显示绿色"连接成功"提示，Token 字段右侧显示绿色对勾图标
  - 保存成功：弹窗关闭，网关列表新增一行，状态显示为"在线"或"离线"

### 编辑网关（弹窗）

- 打开方式：点击列表行 `编辑` 按钮
- 按钮：
  - `测试连接`：使用更新后的信息测试
  - `取消`：关闭弹窗
  - `确认`：保存修改
- 表单字段：
  - 同「新建网关」，所有字段均可修改
- 提示：
  - 页面顶部黄色提示条："修改 Token 后需要等待下次心跳或手动 `测试连接` 验证"
- 行为：
  - 保存成功：弹窗关闭，列表中该网关的地址:端口字段更新为新值

### 生成注册码（弹窗）

- 打开方式：点击列表页 `生成注册码` 按钮
- 按钮：
  - `生成`：生成一次性注册码
- 表单字段：
  - `网关名称`：必填
  - `过期时间`：默认 10 分钟，可调整为 5/10/30 分钟
- 展示内容：
  - `注册码`：注册码 + `复制` 按钮
  - `过期时间`：显示具体到期时刻
  - `使用说明`：文案"将此注册码填入 Node-RED 侧插件完成注册"
- 状态：
  - 注册码过期时显示"已过期"灰色提示，禁用复制按钮
  - 网关使用注册码完成注册后，弹窗自动关闭，列表出现新网关，状态显示为"在线"

### 删除确认（气泡）

- 打开方式：点击列表行 `删除` 按钮
- 按钮：
  - `取消`：关闭气泡
  - `确定`：执行删除
- 文案：
  - "确定删除网关「XXX」吗？"
  - "该网关下 N 个设备将转为未绑定状态。"
- 行为：
  - 删除成功：网关从列表中消失；其下已下发设备的网关绑定关系解除，设备状态变更为"未绑定"，设备记录本身仍存在
