# 设备模型管理

管理所有设备类型模板，支持创建、编辑、复制、导入点位、状态切换、删除、查看版本历史，定义设备基本信息（厂商、型号、协议）和点位模型（点位名称、地址、数据类型、采集频率等）。

## 通用交互与约束

- 权限控制：运维人员、设备管理人员可查看；仅运维人员可执行编辑、删除、新建、状态切换
- 列表：默认按 `创建时间` 倒序排列
- 新建/编辑：必填项缺失时字段下方红字提示"请输入 XXX"；点位编码重复时红框提示"该编码已存在"；采集频率超出范围时红字提示"采集频率需在 100-3600000ms 之间"；保存按钮点击后置灰，请求返回后恢复
- 删除：弹出二次确认气泡，文字"确定删除模型「XXX」吗？"；若模型已被设备实例使用，气泡中额外显示红色提示"该模型已被 X 个设备实例使用，无法删除"，`确定` 按钮置灰
- 状态切换：停用操作弹出确认气泡，文字"确定停用模型「XXX」吗？停用后将不能用于创建新的设备实例。"
- 导入预览：上传文件后先显示预览列表，区分成功行和失败行，失败行标注原因

## 【设备模型管理】产品页面功能描述及交互描述

### 说明

- 模型修改后版本号递增（如 v1.0 → v1.1），已使用该模型的设备实例保留旧版本，不自动更新
- 禁止删除已被设备实例使用的模型，必须先解除所有引用
- 停用的模型不能用于创建新的设备实例，但已创建的实例不受影响
- 同一模型内点位编码必须唯一
- 采集频率范围为 100ms-3600000ms（100毫秒到1小时），默认 1000ms
- 数据类型支持：`int`、`float`、`string`、`bool`、`date`、`datetime`、`json`
- 读写权限支持：`read`、`write`、`readWrite`
- 协议类型从预定义列表选择：`Modbus TCP`、`S7`、`OPC UA`、`MQTT`、`HTTP`、`CoAP`、`MTConnect`、`FOCAS`

### 模型列表页

- 搜索条件：
  - `模型名称`：输入框，模糊匹配，默认空
  - `协议类型`：下拉选，选项 `全部` + 预定义协议列表，默认 `全部`
  - `状态`：下拉选，选项 `全部` / `启用` / `停用`，默认 `全部`
- 按钮：
  - `查询`：更新列表数据，重置页码为 1
  - `重置`：清空所有筛选条件
  - `新建模型`：打开 `新建设备模型（弹窗）`
- 列表展示：
  - 字段：`模型名称`（文本，最多展示 1 行，超长省略号）、`厂商`、`型号`、`协议`（文本，显示协议名称）、`点位数量`（数字，格式"X 个点位"）、`状态`（标签，绿色=启用，灰色=停用）、`版本号`（文本，格式"vX.Y"）、`实例使用数`（数字，格式"已使用 X 次"）、`创建时间`（文本，格式"YYYY-MM-DD HH:mm"）
- 行内操作：
  - `编辑`：打开 `编辑设备模型（弹窗）`，回显当前数据
  - `复制`：打开 `复制设备模型（弹窗）`，预填原模型数据
  - `导入点位`：打开 `导入点位（弹窗）`
  - `启用/停用`：切换模型状态
  - `删除`：弹出二次确认气泡，确认后删除
  - `版本历史`：打开 `版本历史（弹窗）`
- 批量操作：
  - `批量停用`：勾选多个启用状态的模型后显示，点击弹出确认气泡

### 新建设备模型（弹窗）

- 打开方式：点击列表页 `新建模型` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：保存模型
- 基本信息区：
  - `模型名称`：必填，1-50 字符
  - `厂商`：必填，1-50 字符
  - `型号`：必填，1-50 字符
  - `协议`：必填，下拉选，选项为预定义协议列表
  - `描述`：选填，最多 200 字符
- 点位列表区：
  - 初始状态：空列表，显示"暂无点位，请添加"
  - 添加点位：点击 `添加点位` 按钮，在列表末尾新增一行
  - 点位字段：
    - `点位名称`：必填，1-50 字符
    - `点位编码`：必填，1-30 字符，同一模型内唯一
    - `数据类型`：下拉选，选项 `int` / `float` / `string` / `bool` / `date` / `datetime` / `json`
    - `读写权限`：下拉选，选项 `read` / `write` / `readWrite`
    - `寄存器地址`：必填，1-100 字符
    - `数据偏移`：选填，整数，默认 0
    - `数据长度`：选填，整数，默认根据数据类型自动设置
    - `采集频率(ms)`：必填，整数，范围 100-3600000，默认 1000
    - `超时时间(ms)`：选填，整数，默认 3000
    - `单位`：选填，1-20 字符
    - `转换公式`：选填，支持简单算术表达式，如"value * 0.1 + 5"
    - `告警阈值(下限)`：选填，数字
    - `告警阈值(上限)`：选填，数字
  - 删除点位：每行末尾 `删除` 按钮
  - 导入点位：点击 `导入点位` 按钮，打开 `导入点位（弹窗）`
- 校验规则：
  - 必填项缺失：字段下方红字提示"请输入 XXX"
  - 点位编码重复：红框提示"该编码已存在"
  - 采集频率超出范围：红字提示"采集频率需在 100-3600000ms 之间"
- 行为：保存成功后弹窗关闭，模型创建成功，状态为"启用"，版本号为 v1.0

### 编辑设备模型（弹窗）

- 打开方式：点击列表行 `编辑` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：保存并创建新版本
- 基本信息区：同「新建设备模型」，所有字段均可修改
- 点位列表区：同「新建设备模型」，支持新增、编辑、删除点位
- 版本提示：弹窗顶部黄色提示条："修改将创建新版本，已使用该模型的设备实例不会自动更新"
- 行为：保存成功后弹窗关闭，模型版本号递增（如 v1.0 → v1.1）

### 复制设备模型（弹窗）

- 打开方式：点击列表行 `复制` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：创建新模型，版本号为 v1.0
- 表单字段：
  - `模型名称`：必填，预填原模型名称 + "_副本"
  - `厂商`：预填原模型值，可修改
  - `型号`：预填原模型值，可修改
  - `协议`：预填原模型值，可修改
  - `描述`：预填原模型值，可修改
- 点位处理：自动复制原模型的所有点位；复制后点位编码不变
- 行为：保存成功后弹窗关闭，列表中新增模型，版本号为 v1.0

### 导入点位（弹窗）

- 打开方式：点击编辑/新建弹窗中的 `导入点位` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `导入`：导入所有成功行的点位
- 操作区：
  - 文件上传：`选择文件` 按钮，支持 CSV/Excel
  - 模板下载：`下载导入模板` 按钮
- 预览区：
  - 上传成功后显示预览列表：成功行（绿色标记，显示将导入的点位信息）、失败行（红色标记，显示失败原因，如"点位编码已存在"、"数据类型错误"）
  - 统计："共 X 行，成功 X 行，失败 X 行"
- 导入模板格式：
  - 字段列表：`点位名称`、`点位编码`、`数据类型`、`读写权限`、`寄存器地址`、`数据偏移`、`数据长度`、`采集频率(ms)`、`超时时间(ms)`、`单位`、`转换公式`、`告警阈值(下限)`、`告警阈值(上限)`

### 版本历史（弹窗）

- 打开方式：点击列表行 `版本历史` 按钮
- 列表展示：
  - 字段：`版本号`、`更新时间`、`更新人`、`变更说明`
  - 排序：按版本号倒序
- 操作：查看详情，点击版本号，展开显示该版本的完整点位列表

### 删除确认（气泡）

- 打开方式：点击列表行 `删除` 按钮
- 按钮：
  - `取消`：关闭气泡
  - `确定`：执行删除
- 文案：
  - "确定删除模型「XXX」吗？"
- 特殊提示：若模型已被设备实例使用，气泡中额外显示红色提示"该模型已被 X 个设备实例使用，无法删除"，`确定` 按钮置灰
