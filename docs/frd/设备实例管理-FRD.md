# 设备实例管理

管理所有设备实例，支持创建、编辑、下发、同步、删除等操作，从设备模型快速创建设备实例（支持批量），实例自动继承模型点位并可添加自定义点位，实时显示每个实例的运行状态。

## 通用交互与约束

- 权限控制：运维人员、设备管理人员可查看；仅运维人员可执行创建、编辑、下发、删除
- 列表：默认按 `创建时间` 倒序排列；实例状态每 5 秒自动刷新
- 新建/编辑：必填项缺失时字段红框 + 下方红字提示；保存按钮点击后置灰，请求返回后恢复
- 删除：两步删除，已下发的实例弹出确认气泡"确定解除该实例的下发配置吗？解除后实例状态变为'待同步'。"；解除后再次点击"删除"弹出二次确认气泡"确定删除实例「XXX」吗？删除后不可恢复。"
- 状态切换：四态（`未绑定` / `待同步` / `运行中` / `离线`），状态颜色依次为灰色 / 黄色 / 绿色 / 红色
- 下发操作：点击 `下发配置` 后按钮置灰，显示"下发中..."；成功后绿色提示"下发成功"，状态变为"运行中"；失败时红色提示失败原因
- 网关联动：网关离线时，该网关上的所有设备实例状态自动变为"离线"；网关恢复后自动变为"运行中"

## 【设备实例管理】产品页面功能描述及交互描述

### 说明

- 实例从模型继承的点位与模型保持关联，模型变更后自动同步到实例
- 实例只能添加自定义点位，不能修改或删除继承的点位
- 设备实例状态：未绑定 → 待同步 → 运行中 → 离线，运行中 ↔ 离线可自动切换
- 网关离线时，该网关上的所有设备实例状态自动变为"离线"；网关恢复后自动变为"运行中"
- 已下发的设备实例删除需两步：先解除下发 → 再删除
- 只有启用状态的设备模型可用于创建设备实例
- 支持批量创建、批量下发配置
- 同一实例内点位编码唯一（包括继承和自定义点位）

### 实例列表页

- 搜索条件：
  - `实例名称`：输入框，模糊匹配，默认空
  - `所属模型`：下拉选，选项 `全部` + 所有模型名称，默认 `全部`
  - `绑定网关`：下拉选，选项 `全部` + 所有网关名称，默认 `全部`
  - `状态`：下拉选，选项 `全部` / `未绑定` / `待同步` / `运行中` / `离线`，默认 `全部`
- 按钮：
  - `查询`：更新列表数据，重置页码为 1
  - `重置`：清空所有筛选条件
  - `新建实例`：打开 `新建设备实例（弹窗）`
  - `批量新建`：打开 `批量新建设备实例（弹窗）`
- 列表展示：
  - 字段：`实例名称`（文本，最多展示 1 行，超长省略号）、`所属模型`（文本，显示模型名称）、`绑定网关`（文本，显示网关名称；未绑定时显示"未绑定"）、`设备地址`（文本，如"192.168.1.100"或"COM3"）、`状态`（标签，灰色=未绑定，黄色=待同步，绿色=运行中，红色=离线）、`点位数量`（文本，格式"继承 X / 自定义 Y"）、`创建时间`（文本，格式"YYYY-MM-DD HH:mm"）、`最近修改时间`（文本，格式"YYYY-MM-DD HH:mm"）
- 行内操作：
  - `编辑`：打开 `编辑设备实例（弹窗）`
  - `更改网关`：打开 `更改网关（弹窗）`
  - `下发配置`：下发实例配置到网关（仅"待同步"状态显示）
  - `解除下发`：停止网关侧采集（仅"运行中"/"离线"状态显示）
  - `同步模型点位`：手动触发从模型同步最新点位（用于自动同步失败场景）
  - `查看点位`：打开 `查看点位（弹窗）`
  - `删除`：两步删除操作
- 批量操作：
  - `批量下发`：勾选多个"待同步"状态的实例后显示，点击弹出确认气泡
  - `批量解除下发`：勾选多个"运行中"或"离线"状态的实例后显示

### 新建设备实例（弹窗）

- 打开方式：点击列表页 `新建实例` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：创建实例
- 表单字段：
  - `所属模型`：必填，下拉选，仅显示启用状态的模型；选择后自动显示模型的点位数量
  - `绑定网关`：选填，下拉选，显示所有网关名称；可选"暂不绑定"
  - `设备地址`：必填，1-100 字符
  - `实例名称`：必填，1-50 字符
  - `描述`：选填，最多 200 字符
- 点位预览区：
  - 选择模型后，自动显示该模型的点位列表预览（只读）
  - 显示：点位名称、编码、数据类型、寄存器地址、采集频率
- 行为：保存成功后弹窗关闭，实例创建成功，状态为"待同步"（如已选网关）或"未绑定"（如未选网关）

### 批量新建设备实例（弹窗）

- 打开方式：点击列表页 `批量新建` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：批量创建所有可创建的实例
- 表单字段：
  - `所属模型`：必填，下拉选（同单个创建）
  - `绑定网关`：选填，下拉选（同单个创建）
  - `设备列表`：多行输入框，每行格式为"设备地址,实例名称"；也可只填地址，名称自动生成
  - `命名规则`：选填，可设置自动生成名称的前缀和起始编号（如"PLC-{001}"）
- 预览区：
  - 解析输入的设备列表，显示预览表格
  - 每行显示：序号、设备地址、实例名称、状态（可创建/已存在/格式错误）
- 行为：保存成功后弹窗关闭，列表新增对应实例

### 编辑设备实例（弹窗）

- 打开方式：点击列表行 `编辑` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：保存修改
- 基本信息区：
  - `实例名称`：可修改
  - `设备地址`：可修改
  - `描述`：可修改
  - `所属模型`：只读，不可修改
  - `绑定网关`：只读（通过 `更改网关` 操作修改）
- 自定义点位区：
  - 列表显示现有自定义点位
  - `添加自定义点位` 按钮：在列表末尾新增一行，字段同模型点位
  - 自定义点位可修改、可删除
  - 继承的点位在此区域只读显示（不可修改/删除），用"继承"标签标识

### 更改网关（弹窗）

- 打开方式：点击列表行 `更改网关` 按钮
- 按钮：
  - `取消`：关闭弹窗
  - `保存`：更新绑定网关
- 表单字段：
  - `当前网关`：只读，显示当前绑定的网关名称
  - `新网关`：必填，下拉选，从已注册网关中选择
- 提示：顶部黄色提示条："更改网关后，如需在新网关上运行，请手动执行'下发配置'。"

### 下发配置 / 解除下发（气泡）

- 打开方式：点击列表行 `下发配置` 或 `解除下发` 按钮
- 按钮：
  - `取消`：关闭气泡
  - `确定`：执行对应操作
- 文案：
  - 下发配置："确定下发配置到网关「XXX」吗？"
  - 解除下发："确定解除该实例的下发配置吗？解除后实例状态变为"待同步"。"
  - 批量下发："确定批量下发选中的 X 个实例配置吗？"

### 同步模型点位（气泡）

- 打开方式：点击列表行 `同步模型点位` 按钮
- 按钮：
  - `取消`：关闭气泡
  - `确定`：执行同步
- 文案：
  - "将从模型「XXX」同步最新点位配置。"
  - "继承点位将更新为模型最新版本，自定义点位不受影响。"

### 查看点位（弹窗）

- 打开方式：点击列表行 `查看点位` 按钮
- 列表展示：
  - 字段：`点位名称`、`点位编码`、`数据类型`、`读写权限`、`寄存器地址`、`采集频率(ms)`、`单位`、`来源`（标签：继承 / 自定义）
  - 支持按"来源"筛选切换查看：全部 / 仅继承 / 仅自定义
  - 排序：按点位编码正序排列

### 删除确认（气泡）

- 打开方式：点击列表行 `删除` 按钮
- 按钮：
  - `取消`：关闭气泡
  - `确定`：执行删除
  - `解除下发`：第一步提示按钮，触发解除下发操作
- 文案：
  - 第一步提示（已下发的实例）："该实例当前处于"运行中"状态，需先解除下发。"
  - 第二步提示："确定删除实例「XXX」吗？删除后不可恢复。"
