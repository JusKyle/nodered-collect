- `frontend npm run lint` 仍有大量既有 lint 问题，主要分布在设备实例、网关、store 等既有代码；本次已保证 `frontend npm run build` 通过。
- 点位导出当前实现为 CSV 文件流；任务说明中写 Excel/CSV，未新增 `xlsx` 依赖。
