## API 接口

| 模块 | 路径 | 方法 | 说明 |
|------|------|------|------|
| 网关管理 | `/api/gateways` | GET | 获取网关列表 |
| | | POST | 创建网关 |
| | `/api/gateways/:id` | GET | 获取网关详情 |
| | | PUT | 更新网关 |
| | | DELETE | 删除网关 |
| | `/api/gateways/test-connection` | POST | 测试连接 |
| 设备模型 | `/api/device-models` | GET | 获取模型列表 |
| | | POST | 创建模型 |
| | `/api/device-models/:id` | GET | 获取模型详情 |
| | | PUT | 更新模型 |
| | | DELETE | 删除模型 |
| 设备实例 | `/api/device-instances` | GET | 获取实例列表 |
| | | POST | 创建实例 |
| | `/api/device-instances/batch` | POST | 批量创建 |
| | `/api/device-instances/:id` | GET | 获取实例详情 |
| | | PUT | 更新实例 |
| | | DELETE | 删除实例 |
| | `/api/device-instances/gateway/:gatewayId` | GET | 按网关查询 |
| 同步记录 | `/api/sync` | GET | 获取同步记录 |
| | | POST | 创建同步记录 |
| | `/api/sync/gateway/:gatewayId` | GET | 按网关查询 |
| | `/api/sync/deploy` | POST | 配置下发 |
| 注册 | `/api/registration/generate` | POST | 生成注册码 |
| | `/api/registration/register` | POST | 注册网关 |

## 启动方式

### 开发环境

**后端**:
```bash
cd backend
npm install
cp .env.example .env
npx prisma migrate dev
npm run dev
```

**前端**:
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

### Docker 部署

```bash
docker-compose up -d
```
